🔧 如何启用Background Fetch

Background Fetch是实验性功能，需要手动开启：

  • 1️⃣ 使用Chrome 74+ 或 Edge 74+
  • 2️⃣ 在地址栏输入 chrome://flags/#enable-experimental-web-platform-features
  • 3️⃣ 将"Experimental Web Platform features"设置为Enabled
  • 4️⃣ 重启浏览器
  • 5️⃣ 使用HTTPS协议访问本页面（localhost除外）
