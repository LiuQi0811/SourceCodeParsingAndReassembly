API 简介

Page Visibility API 让开发者能够确定页面当前是否可见。这对于优化性能和用户体验非常有用。

// 监听可见性变化 document.addEventListener('visibilitychange', () => { if (document.hidden) { pauseVideo(); // 页面隐藏 } else { resumeVideo(); // 页面可见 } });

API 支持检测

API 类型 描述 状态

属性说明

属性 类型 可能值
document.visibilityState string visible, hidden, prerender
document.hidden boolean true, false

visibilityState 值

值 hidden 触发条件
visible false 页面在前台显示
hidden true 切换标签/最小化
prerender true 页面预渲染中

使用示例

// 基本用法 document.addEventListener('visibilitychange', () => { console.log(document.visibilityState); }); // 视频自动暂停 const video = document.querySelector('video'); document.addEventListener('visibilitychange', () => { document.hidden ? video.pause() : video.play(); });
