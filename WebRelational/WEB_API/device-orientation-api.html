DeviceOrientationEvent Interface

DeviceOrientationEvent.alpha Property

Z-axis rotation in degrees [0, 360]. Represents the rotation around the axis perpendicular to the device screen.

DeviceOrientationEvent.beta Property

X-axis rotation in degrees [-180, 180]. Represents the rotation from front to back.

DeviceOrientationEvent.gamma Property

Y-axis rotation in degrees [-90, 90]. Represents the rotation from left to right.

DeviceOrientationEvent.absolute Property

Boolean indicating if the orientation data is absolute (relative to Earth's coordinate system).

DeviceOrientationEvent.webkitCompassHeading Property

The compass heading in degrees (WebKit specific). Available on iOS devices.

DeviceOrientationEvent.requestPermission() Method

Requests permission to receive device orientation events. Required on iOS 13+.

DeviceMotionEvent Interface

DeviceMotionEvent.acceleration Property

Acceleration without gravity (x, y, z in m/s²). May be null if not available.

DeviceMotionEvent.accelerationIncludingGravity Property

Acceleration including gravity (x, y, z in m/s²).

DeviceMotionEvent.rotationRate Property

Rotation rate (alpha, beta, gamma in degrees/second). May be null if not available.

DeviceMotionEvent.interval Property

The interval at which data is obtained from the hardware (in milliseconds).

DeviceMotionEvent.requestPermission() Method

Requests permission to receive device motion events. Required on iOS 13+.

Events

deviceorientation Event

Fired when the device orientation changes significantly.

deviceorientationabsolute Event

Fired when absolute device orientation is available.

devicemotion Event

Fired when device motion data is available.

Usage Example

// Check if API is supported if (window.DeviceOrientationEvent) { // Request permission on iOS 13+ if (typeof DeviceOrientationEvent.requestPermission === 'function') { DeviceOrientationEvent.requestPermission() .then(permission => { if (permission === 'granted') { window.addEventListener('deviceorientation', handler); } }); } else { // Non-iOS or older iOS window.addEventListener('deviceorientation', handler); } } function handler(event) { console.log('Alpha:', event.alpha); // Z-axis (0-360) console.log('Beta:', event.beta); // X-axis (-180 to 180) console.log('Gamma:', event.gamma); // Y-axis (-90 to 90) console.log('Absolute:', event.absolute); } // Device Motion if (window.DeviceMotionEvent) { window.addEventListener('devicemotion', (event) => { console.log('Acceleration:', event.acceleration); console.log('Acceleration with gravity:', event.accelerationIncludingGravity); console.log('Rotation rate:', event.rotationRate); console.log('Interval:', event.interval); }); }
