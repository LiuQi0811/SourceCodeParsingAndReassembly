URL 构造函数

URL new URL(url, base?)

创建一个 URL 对象，解析 URL 字符串。

const url = new URL('https://example.com/path?name=value#hash'); console.log(url.hostname); // 'example.com' console.log(url.pathname); // '/path' console.log(url.searchParams.get('name')); // 'value'

URL 属性

属性说明示例
href完整 URLhttps://example.com/path?q=1#hash
origin协议 + 主机 + 端口https://example.com:8080
protocol协议https:
host主机名 + 端口example.com:8080
hostname主机名example.com
port端口8080
pathname路径/path/to/page
search查询字符串?name=value
hash哈希#section

URLSearchParams

PARAMS new URLSearchParams(init?)

创建 URLSearchParams 对象。

const params = new URLSearchParams('name=John&age=30'); params.append('city', 'NYC'); params.set('age', '31'); console.log(params.get('name')); // 'John' console.log(params.toString()); // 'name=John&age=31&city=NYC'
方法说明
append(name, value)追加参数
delete(name)删除参数
get(name)获取参数值
getAll(name)获取所有同名参数值
has(name)检查参数是否存在
set(name, value)设置参数（覆盖）
sort()按键名排序
toString()返回查询字符串
forEach(callback)遍历参数
entries() / keys() / values()迭代器

Blob URL

BLOB URL.createObjectURL(blob)

创建 Blob URL，用于引用 Blob 或 File 对象。

const blob = new Blob(['Hello World'], { type: 'text/plain' }); const url = URL.createObjectURL(blob); console.log(url); // 'blob:https://example.com/uuid' // 使用后释放 URL.revokeObjectURL(url);
