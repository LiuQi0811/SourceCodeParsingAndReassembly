MediaList

MediaList represents a list of media queries. It's accessed via stylesheet.media or style/media elements.

// MediaList properties and methods const media = stylesheet.media; media.mediaText; // 'screen, print' media.length; // 2 media.item(0); // 'screen' media.appendMedium('tv'); media.deleteMedium('print');

CSS Interface

The CSS interface provides utility methods for CSS operations.

// CSS.supports() checks if a CSS property-value is supported CSS.supports('display', 'grid'); // true/false CSS.supports('(display: grid)'); // true/false // CSS.escape() escapes a string for use as CSS identifier CSS.escape('my-element'); // 'my-element' CSS.escape('1foo'); // '\\31 foo'
adoptedStyleSheets

adoptedStyleSheets allows sharing stylesheets across documents and shadow roots.

// adoptedStyleSheets usage const sheet = new CSSStyleSheet(); sheet.replaceSync('.my-class { color: red; }'); document.adoptedStyleSheets = [...document.adoptedStyleSheets, sheet];

StyleSheet Interface

stylesheet.type Property

Returns the MIME type (usually "text/css").

stylesheet.href Property

Returns the URL of the stylesheet, or null for inline styles.

stylesheet.ownerNode Property

Returns the Element that links to the stylesheet.

stylesheet.disabled Property

Gets or sets whether the stylesheet is disabled.

stylesheet.media Property

Returns the MediaList for the stylesheet.

CSSStyleSheet Interface

stylesheet.cssRules Property

Returns a CSSRuleList of all CSS rules in the stylesheet.

stylesheet.insertRule(rule, index) Method

Inserts a new rule at the specified index. Returns the index.

stylesheet.deleteRule(index) Method

Deletes the rule at the specified index.

stylesheet.replace(text) Method

Replaces the stylesheet content asynchronously. Returns Promise.

stylesheet.replaceSync(text) Method

Replaces the stylesheet content synchronously.

CSSRule Interface

rule.type Property

Returns the type constant (1=StyleRule, 3=ImportRule, 4=MediaRule, etc.).

rule.cssText Property

Returns the CSS text of the rule.

rule.parentStyleSheet Property

Returns the parent CSSStyleSheet.

rule.parentRule Property

Returns the parent CSSRule if nested.

CSSStyleDeclaration Interface

style.cssText Property

Gets or sets the entire CSS text.

style.length Property

Returns the number of properties set.

style.setProperty(property, value, priority) Method

Sets a CSS property value with optional priority.

style.getPropertyValue(property) Method

Returns the value of a CSS property.

style.getPropertyPriority(property) Method

Returns the priority ("important" or "").

style.removeProperty(property) Method

Removes a CSS property. Returns the old value.

style.item(index) Method

Returns the property name at the given index.

MediaList Interface

media.mediaText Property

Gets or sets the media query text.

media.length Property

Returns the number of media queries.

media.item(index) Method

Returns the media query at the given index.

media.appendMedium(medium) Method

Adds a media query to the list.

media.deleteMedium(medium) Method

Removes a media query from the list.

CSS Interface

CSS.supports(property, value) Static Method

Checks if a CSS property-value pair is supported.

CSS.supports(conditionText) Static Method

Checks if a CSS condition is supported.

CSS.escape(ident) Static Method

Escapes a string for use as a CSS identifier.
