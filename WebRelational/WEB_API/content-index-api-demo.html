Content Index API Explorer

Content Index API 概览

Content Index API 允许 PWA 将内容添加到浏览器的内容索引中，使用户可以离线浏览

➕

添加内容

使用 add() 方法将内容添加到索引

📋

获取列表

使用 getAll() 获取所有已索引内容

🗑️

删除内容

使用 delete() 从索引中移除内容

📱

离线访问

用户可在浏览器内容索引中离线访问

📊 API 支持检测

API 状态 说明

⚡ 快速开始

// 获取 Service Worker Registration const registration = await navigator.serviceWorker.ready; // 添加内容到索引 await registration.index.add({ id: 'article-1', title: '我的文章', description: '这是一篇精彩的文章', url: '/articles/1', category: 'article' }); // 获取所有索引内容 const contents = await registration.index.getAll(); // 删除内容 await registration.index.delete('article-1');

⚠️ 使用要求

⚠️
前置条件
Content Index API 需要以下条件才能使用：
  • 必须注册 Service Worker
  • 必须通过 HTTPS 提供页面
  • PWA 需要可安装
  • 内容 URL 需要在 Cache 中可用

ContentIndex 接口

ContentIndex 接口提供管理内容索引的方法

📍 访问方式

registration.index ContentIndex
通过 ServiceWorkerRegistration 访问

🔧 方法

方法 参数 返回值 说明
add() ContentDescription Promise<void> 添加内容到索引
delete() id: string Promise<void> 从索引删除内容
getAll() 无 Promise<ContentDescription[]> 获取所有索引内容

ContentDescription 接口

ContentDescription 描述索引中的内容项

📋 属性

属性 类型 必需 说明
id string ✓ 内容唯一标识符
title string ✓ 内容标题
description string ✓ 内容描述
url string ✓ 内容 URL
icons ImageResource[] 图标数组
category string 内容类别

🏷️ 类别值

值 说明
"" (空)默认类别
"homepage"主页内容
"article"文章内容
"video"视频内容
"audio"音频内容

📝 Service Worker 代码

// service-worker.js // 安装事件 self.addEventListener('install', (event) => { self.skipWaiting(); }); // 激活事件 self.addEventListener('activate', (event) => { event.waitUntil(self.clients.claim()); }); // 监听内容删除事件 self.addEventListener('contentdelete', (event) => { console.log('Content deleted:', event.id); }); // 缓存策略 self.addEventListener('fetch', (event) => { event.respondWith( caches.match(event.request) .then(response => response || fetch(event.request)) ); });
