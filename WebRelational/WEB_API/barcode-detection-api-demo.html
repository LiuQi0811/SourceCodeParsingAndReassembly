API Introduction

The Barcode Detection API detects linear and two-dimensional barcodes in images. It provides a fast and efficient way to read barcode data without requiring external libraries.

🎯 Core Features

  • ✓ Detect multiple barcode formats
  • ✓ Support for 1D and 2D barcodes
  • ✓ Hardware-accelerated detection
  • ✓ Works with images and video
  • ✓ Returns bounding box and corner points
  • ✓ No external dependencies required

📦 API Components

BarcodeDetector Interface
BarcodeDetector() Constructor
getSupportedFormats() Static
detect() Method
DetectedBarcode Interface

💻 Quick Start

// Check for API support if ('BarcodeDetector' in window) { // Create a barcode detector const detector = new BarcodeDetector({ formats: ['qr_code', 'ean_13'] }); // Detect barcodes in an image const barcodes = await detector.detect(imageElement); // Process detected barcodes barcodes.forEach(barcode => { console.log('Format:', barcode.format); console.log('Value:', barcode.rawValue); console.log('Bounds:', barcode.boundingBox); }); }

🔍 BarcodeDetector Interface

Experimental

The BarcodeDetector interface of the Barcode Detection API allows detection of linear and two dimensional barcodes in images.

Web IDL Definition

[Exposed=(Window,Worker)] interface BarcodeDetector { constructor(optional BarcodeDetectorOptions options = {}); static Promise<sequence<BarcodeFormat>> getSupportedFormats(); Promise<sequence<DetectedBarcode>> detect(ImageBitmapSource image); }; dictionary BarcodeDetectorOptions { sequence<BarcodeFormat> formats; };

🏗️ Constructor

new BarcodeDetector(); new BarcodeDetector(options);

Options

formats Array of barcode formats to detect (optional)

⚙️ Static Methods

getSupportedFormats() Promise<string[]>

Returns a Promise which fulfills with an Array of supported barcode format types.

🔍 detect() Method

Method

Detects barcodes in the given image. Returns a Promise that resolves with an array of DetectedBarcode objects.

Parameters

Parameter Type Description
image ImageBitmapSource An image to detect barcodes in (HTMLImageElement, SVGImageElement, HTMLVideoElement, HTMLCanvasElement, ImageBitmap, etc.)

Return Value

Promise<DetectedBarcode[]> Promise

💻 Examples

// Create detector with specific formats const detector = new BarcodeDetector({ formats: ['qr_code', 'ean_13', 'code_128'] }); // Detect from an image element const image = document.getElementById('myImage'); const barcodes = await detector.detect(image); // Process results barcodes.forEach(barcode => { console.log(`Found ${barcode.format}: ${barcode.rawValue}`); }); // Detect from video stream const video = document.getElementById('video'); async function scanFrame() { const barcodes = await detector.detect(video); if (barcodes.length > 0) { console.log('Detected:', barcodes[0].rawValue); } requestAnimationFrame(scanFrame); } scanFrame();

📊 DetectedBarcode Interface

Experimental

The DetectedBarcode interface represents a detected barcode. It contains the decoded data, format, and location information.

Web IDL Definition

interface DetectedBarcode { readonly attribute BarcodeFormat format; readonly attribute DOMString rawValue; readonly attribute DOMRectReadOnly boundingBox; readonly attribute FrozenArray<Point2D> cornerPoints; }; dictionary Point2D { double x; double y; };

🏷️ format Property

Read-only

The detected barcode format. Returns one of the supported barcode format strings.

Type BarcodeFormat

📝 rawValue Property

Read-only

A string decoded from the barcode data.

Type DOMString

📐 boundingBox Property

Read-only

A DOMRectReadOnly which returns the dimensions of a rectangle representing the extent of a detected barcode, aligned with the image.

Type DOMRectReadOnly

📍 cornerPoints Property

Read-only

The x and y co-ordinates of the four corner points of the detected barcode relative to the image, starting with the top left and working clockwise.

Type Point2D[]

💻 Examples

// Access detected barcode properties const barcodes = await detector.detect(image); barcodes.forEach(barcode => { // Format: e.g., 'qr_code', 'ean_13' console.log('Format:', barcode.format); // Decoded value console.log('Value:', barcode.rawValue); // Bounding box console.log('Position:', { x: barcode.boundingBox.x, y: barcode.boundingBox.y, width: barcode.boundingBox.width, height: barcode.boundingBox.height }); // Corner points (for perspective distortion) barcode.cornerPoints.forEach((point, i) => { console.log(`Corner ${i}: (${point.x}, ${point.y})`); }); });
