⚡ 快速开始

性能分析

使用 profile 分析代码性能

📈 console.profile() / profileEnd()

⚠️
注意
profile 方法需要在浏览器开发者工具的 Performance 面板中查看结果
// 开始性能分析 console.profile('我的分析'); // 执行需要分析的代码 for (let i = 0; i < 10000; i++) { Math.sqrt(i); } // 结束分析 console.profileEnd('我的分析'); // 在开发者工具 Performance 面板查看结果
