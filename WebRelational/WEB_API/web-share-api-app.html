API 文档

Web Share API 完整文档

核心方法

METHOD navigator.share(data)

调用系统分享界面，返回 Promise。

METHOD navigator.canShare(data)

检查是否可以分享指定数据，返回 boolean。

ShareData 接口

属性类型说明
titlestring分享标题
textstring分享文本
urlstring分享链接
filesFile[]分享文件数组

使用示例

// 分享文本和链接 if (navigator.share) { try { await navigator.share({ title: '分享标题', text: '分享文本内容', url: 'https://example.com' }); console.log('分享成功'); } catch (err) { console.log('分享失败:', err); } } // 分享文件 if (navigator.canShare && navigator.canShare({ files: files })) { await navigator.share({ files: files, title: '文件分享', text: '查看这些文件' }); } // 检查分享能力 if (navigator.canShare({ title: 'test' })) { console.log('支持分享'); }
