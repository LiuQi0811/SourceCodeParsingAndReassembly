API 文档

WebVR API 完整文档

⚠️ 废弃警告：WebVR API 已被 WebXR Device API 取代。建议新项目使用 WebXR。

核心接口

INTERFACE Navigator.vr

WebVR API 的主入口。

INTERFACE VRDisplay

代表一个 VR 显示设备。

INTERFACE VRDisplayCapabilities

VR 显示设备的能力描述。

INTERFACE VRPose

VR 设备的姿态信息（位置和方向）。

INTERFACE VREyeParameters

眼睛渲染参数。

INTERFACE VRFrameData

每帧渲染所需的数据。

INTERFACE VRLayer

VR 渲染图层。

主要方法

方法说明
navigator.getVRDisplays()获取 VR 设备列表
VRDisplay.requestPresent(layers)请求进入 VR 模式
VRDisplay.exitPresent()退出 VR 模式
VRDisplay.getFrameData(frameData)获取帧数据
VRDisplay.submitFrame()提交渲染帧
VRDisplay.getEyeParameters(eye)获取眼睛参数
VRDisplay.getPose()获取当前姿态

替代方案

WebXR Device API 是 WebVR 的替代方案，提供更现代的 AR/VR 支持。
  • navigator.xr.requestDevice()
  • XRSession, XRFrame, XRView
  • 支持 AR 和 VR 模式
