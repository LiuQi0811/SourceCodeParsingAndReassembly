Manifest 配置指南

在 manifest.json 中配置 launch_handler 以控制启动行为。 

{
  "launch_handler": {
    "client_mode": [
      "navigate-existing",
      "auto"
    ]
  },
  "protocol_handlers": [{
    "protocol": "web+demo",
    "url": "/?from=%s"
  }]
}
