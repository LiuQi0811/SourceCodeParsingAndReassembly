CookieStore Interface

cookieStore.get(name) Method

Returns a Promise that resolves with the first cookie matching the given name.

cookieStore.get(options) Method

Returns a Promise that resolves with the first cookie matching the given options (name, url).

cookieStore.getAll(name?) Method

Returns a Promise that resolves with an array of all cookies, or cookies matching the name.

cookieStore.set(name, value) Method

Sets a cookie with the given name and value. Returns a Promise.

cookieStore.set(options) Method

Sets a cookie with the given options (name, value, domain, path, expires, maxAge, secure, sameSite).

cookieStore.delete(name) Method

Deletes the cookie with the given name. Returns a Promise.

cookieStore.delete(options) Method

Deletes cookies matching the given options (name, domain, path).

CookieListItem Interface

name Property

The name of the cookie.

value Property

The value of the cookie.

domain Property

The domain the cookie is valid for.

path Property

The path the cookie is valid for.

expires Property

The expiration time as a Unix timestamp (milliseconds).

secure Property

Whether the cookie requires HTTPS.

sameSite Property

The SameSite attribute: "strict", "lax", or "none".

CookieStoreSetOptions Interface

name Property

The name of the cookie. Required.

value Property

The value of the cookie. Required.

domain Property

The domain to scope the cookie to. Optional.

path Property

The path to scope the cookie to. Optional.

expires Property

Expiration time as Date or timestamp. Optional.

maxAge Property

Max age in seconds. Optional.

secure Property

Whether cookie requires HTTPS. Optional.

sameSite Property

SameSite attribute. Optional.

CookieChangeEvent Interface

event.type Property

The event type ("change").

event.changed Property

Array of cookies that were added or modified.

event.deleted Property

Array of cookies that were deleted.

SameSite Enum

"strict" Value

Cookie only sent in first-party context.

"lax" Value

Cookie sent with top-level navigations.

"none" Value

Cookie sent in all contexts (requires secure=true).

Usage Example

// Check if API is supported if ('cookieStore' in window) { // Set a cookie await cookieStore.set({ name: 'session', value: 'abc123', path: '/', secure: true, sameSite: 'lax' }); // Get a cookie const cookie = await cookieStore.get('session'); console.log(cookie.value); // 'abc123' // Get all cookies const allCookies = await cookieStore.getAll(); console.log(allCookies); // Delete a cookie await cookieStore.delete('session'); // Listen for changes cookieStore.addEventListener('change', (event) => { console.log('Changed:', event.changed); console.log('Deleted:', event.deleted); }); }
