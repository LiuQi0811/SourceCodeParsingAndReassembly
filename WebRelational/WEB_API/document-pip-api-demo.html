Document Picture-in-Picture API 概览

Document PiP API 允许创建一个始终置顶的画中画窗口，可在其中显示任意 HTML 内容

📊 API 支持检测

API 状态 说明

⚡ 快速开始

// 检查 API 支持 if ('documentPictureInPicture' in window) { // 请求创建画中画窗口 const pipWindow = await window.documentPictureInPicture.requestWindow({ width: 400, height: 300 }); // 复制样式到画中画窗口 const styles = document.querySelectorAll('style, link[rel="stylesheet"]'); styles.forEach(style => { pipWindow.document.head.appendChild(style.cloneNode(true)); }); // 添加内容到画中画窗口 pipWindow.document.body.innerHTML = `<h1>画中画内容</h1>`; // 监听窗口关闭 pipWindow.addEventListener('pagehide', () => { console.log('画中画窗口已关闭'); }); }

DocumentPictureInPicture 接口

DocumentPictureInPicture 是访问画中画功能的主接口

📍 访问方式

属性 类型 说明
window.documentPictureInPicture DocumentPictureInPicture 访问画中画 API

📋 属性

属性 类型 说明
window Window? 当前画中画窗口引用，无则返回 null
width number 画中画窗口宽度（只读）
height number 画中画窗口高度（只读）

🔧 方法

方法 参数 返回值 说明
requestWindow() options? Promise<Window> 请求创建画中画窗口
