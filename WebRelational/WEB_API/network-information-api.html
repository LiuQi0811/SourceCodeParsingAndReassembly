什么是数据节省模式？

数据节省模式（Data Saver）是浏览器或操作系统提供的一种功能，用于减少网络数据使用量。 当用户启用此模式时，网站应该：

  • 减少或延迟加载非关键资源
  • 使用低分辨率图片
  • 暂停自动播放视频
  • 减少后台数据同步
  • 压缩传输数据
// 检测数据节省模式 const connection = navigator.connection; if (connection.saveData) { // 用户启用了数据节省模式 loadLowResolutionImages(); disableAutoPlay(); }

按流量计费连接

某些网络连接（如移动热点、蜂窝数据）可能按流量计费。 检测此类连接可以帮助应用减少不必要的数据使用。

使用示例

// 获取网络信息 const connection = navigator.connection; // 读取连接类型 console.log(connection.type); // 'wifi' | 'cellular' | 'ethernet' console.log(connection.effectiveType); // '4g' | '3g' | '2g' | 'slow-2g' // 读取带宽信息 console.log(connection.downlink); // 下行带宽 (Mbps) console.log(connection.rtt); // 往返时间 (ms) // 检测数据节省模式 if (connection.saveData) { enableDataSaverMode(); } // 监听网络变化 connection.addEventListener('change', () => { updateNetworkUI(); });
