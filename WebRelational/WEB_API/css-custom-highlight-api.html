The CSS Custom Highlight API provides a mechanism for styling arbitrary text ranges on a document by using JavaScript to create the ranges, and CSS to style them.

This API is useful for creating custom text highlighting effects without modifying the DOM structure. You can use it for search results highlighting, text annotations, spelling error indicators, and more.

The API consists of two main interfaces: Highlight and HighlightRegistry. The Highlight interface represents a collection of Range objects, while the HighlightRegistry (accessed via CSS.highlights) manages all registered highlights.

The CSS Custom Highlight API is a powerful web platform feature that enables developers to create custom text highlighting effects without modifying the underlying DOM structure.

Key benefits include:

  • Non-destructive text styling - Original DOM remains unchanged
  • Performance optimized - Browser handles rendering efficiently
  • Flexible styling - Use any CSS properties supported by ::highlight pseudo-element
  • Multiple ranges - A single highlight can span multiple text ranges

This API is particularly useful for applications like text editors, document viewers, search interfaces, and educational tools where dynamic text highlighting is essential.

The CSS Custom Highlight API represents a significant advancement in web text styling capabilities. This API allows developers to create rich text highlighting experiences without the performance overhead of traditional DOM manipulation methods.

Traditional approaches to text highlighting often involve wrapping text in span elements, which can cause performance issues with large documents and break text selection behavior. The CSS Custom Highlight API solves these problems by operating at a lower level, allowing the browser to optimize rendering.

The API is particularly well-suited for implementing features like:

  • Search result highlighting with customizable styles
  • Real-time spell checking with visual indicators
  • Grammar checking with distinct visual treatment
  • Text annotations and collaborative editing markers
  • Code syntax highlighting in custom editors

Browser support for the CSS Custom Highlight API is growing, with major browsers implementing the specification. Developers should check for API availability before using it in production applications.

Highlight Interface

new Highlight(...ranges) Constructor

Creates a new Highlight object containing the specified Range objects.

...ranges Range[] - Zero or more Range objects to include
highlight.priority Property

Gets or sets the priority of the highlight. Higher values are rendered on top.

highlight.type Property

Gets or sets the type of highlight. Values: 'highlight', 'spelling-error', 'grammar-error'.

highlight.add(range) Method

Adds a new Range to the highlight. Returns the Highlight object.

highlight.clear() Method

Removes all Range objects from the highlight.

highlight.delete(range) Method

Removes the specified Range from the highlight. Returns true if removed.

highlight.has(range) Method

Returns true if the highlight contains the specified Range.

highlight.size Property (readonly)

Returns the number of Range objects in the highlight.

highlight.entries() Method

Returns an iterator for the [range, range] pairs in the highlight.

highlight.forEach(callback) Method

Executes a callback for each Range in the highlight.

highlight.keys() Method

Returns an iterator for the Range objects in the highlight.

highlight.values() Method

Returns an iterator for the Range objects in the highlight.

HighlightRegistry Interface

CSS.highlights Property (static)

Returns the HighlightRegistry object for the document.

registry.set(name, highlight) Method

Registers a Highlight with the given name. Returns the registry.

registry.get(name) Method

Returns the Highlight registered with the given name, or undefined.

registry.has(name) Method

Returns true if a Highlight is registered with the given name.

registry.delete(name) Method

Removes the Highlight registered with the given name. Returns true if removed.

registry.clear() Method

Removes all registered Highlights.

registry.size Property (readonly)

Returns the number of registered Highlights.

CSS ::highlight() Pseudo-element

::highlight(custom-highlight-name) Pseudo-element

Styles the text ranges in the named highlight. Supports properties like background-color, color, text-decoration, etc.

/* Example CSS */ ::highlight(my-highlight) { background-color: rgba(236, 72, 153, 0.3); color: inherit; } ::highlight(spelling-error) { text-decoration: underline wavy red; } ::highlight(grammar-error) { text-decoration: underline wavy blue; }

Complete Example

// Create a Range const range = document.createRange(); const textNode = document.querySelector('p').firstChild; range.setStart(textNode, 0); range.setEnd(textNode, 10); // Create a Highlight const highlight = new Highlight(range); highlight.priority = 1; highlight.type = 'highlight'; // Register the highlight CSS.highlights.set('my-highlight', highlight); // Style with CSS // ::highlight(my-highlight) { background-color: yellow; }
