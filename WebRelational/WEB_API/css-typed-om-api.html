CSSMathValue Operations

CSSMathValue allows mathematical operations on CSS values. Create complex calculations with type safety.

// Result will appear here const result = CSS.px(100).add(CSS.px(50)); // CSSMathSum { ... }
Expression Builder
// CSSMathValue examples // Addition const sum = CSS.px(100).add(CSS.px(50)); // CSSMathSum(100px + 50px) // Multiplication const doubled = CSS.px(100).mul(CSS.number(2)); // CSSMathProduct(100px * 2) // Complex expression const calc = CSS.px(100).add(CSS.vw(10)).sub(CSS.rem(1)); // CSSMathSum(100px + 10vw - 1rem) // Min/Max const min = CSS.min(CSS.px(100), CSS.percent(50)); const max = CSS.max(CSS.px(200), CSS.vw(20)); // Clamp const clamped = CSS.clamp(CSS.px(100), CSS.px(150), CSS.px(200)); // Value between 100px and 200px
