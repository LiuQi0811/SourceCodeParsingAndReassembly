API 文档

Web Crypto API 完整文档

核心对象

OBJECT crypto.subtle

SubtleCrypto 对象，提供所有加密操作方法。

主要方法

方法说明
encrypt(algorithm, key, data)加密数据
decrypt(algorithm, key, data)解密数据
sign(algorithm, key, data)生成签名
verify(algorithm, key, signature, data)验证签名
digest(algorithm, data)计算哈希
generateKey(algorithm, extractable, keyUsages)生成密钥
deriveKey(algorithm, baseKey, derivedKeyType, extractable, keyUsages)派生密钥
deriveBits(algorithm, baseKey, length)派生位
importKey(format, keyData, algorithm, extractable, keyUsages)导入密钥
exportKey(format, key)导出密钥

支持的算法

类别算法
摘要SHA-1, SHA-256, SHA-384, SHA-512
对称加密AES-GCM, AES-CBC, AES-CTR
非对称加密RSA-OAEP, RSA-PSS
签名RSASSA-PKCS1-v1_5, ECDSA
密钥派生PBKDF2, HKDF, ECDH

使用示例

// 生成随机值 const array = new Uint8Array(16); crypto.getRandomValues(array); // 计算哈希 const encoder = new TextEncoder(); const data = encoder.encode('Hello'); const hash = await crypto.subtle.digest('SHA-256', data); // 生成 AES 密钥 const key = await crypto.subtle.generateKey( { name: 'AES-GCM', length: 256 }, true, ['encrypt', 'decrypt'] ); // 加密 const iv = crypto.getRandomValues(new Uint8Array(12)); const encrypted = await crypto.subtle.encrypt( { name: 'AES-GCM', iv }, key, data ); // 解密 const decrypted = await crypto.subtle.decrypt( { name: 'AES-GCM', iv }, key, encrypted );
