Summarizer API 概述

Summarizer API 是一个内置的 AI API，用于生成文本摘要。它可以在设备上运行，无需网络连接，保护用户隐私。

PROPERTY ai.summarizer

检查 Summarizer API 是否可用。

// 检查 API 可用性 if ('ai' in self && 'summarizer' in ai) { const capabilities = await ai.summarizer.capabilities(); console.log('API 可用:', capabilities.available); }
ASYNC ai.summarizer.capabilities()

获取 API 的能力信息。

返回值

available string 'readily' | 'after-download' | 'no'

创建摘要器

ASYNC ai.summarizer.create(options?)

创建一个新的 Summarizer 实例。

Options 参数

type string 摘要类型: 'tl;dr' | 'key-points' | 'teaser' | 'headline'
format string 输出格式: 'plain-text' | 'markdown'
length string 摘要长度: 'short' | 'medium' | 'long'
// 创建摘要器 const summarizer = await ai.summarizer.create({ type: 'tl;dr', format: 'plain-text', length: 'medium' });
ASYNC ai.summarizer.create(options, signal)

使用 AbortSignal 创建摘要器，支持取消操作。

// 支持取消的创建 const controller = new AbortController(); try { const summarizer = await ai.summarizer.create({ type: 'key-points' }, controller.signal); } catch (e) { if (e.name === 'AbortError') { console.log('创建已取消'); } }

生成摘要

ASYNC summarizer.summarize(text, options?)

对输入文本生成摘要。

参数

text string 需要摘要的文本内容
options.signal AbortSignal 用于取消操作的信号
// 生成摘要 const text = `这是一段很长的文本内容...`; const summary = await summarizer.summarize(text); console.log('摘要:', summary);
ASYNC summarizer.summarizeStreaming(text, options?)

流式生成摘要，返回 ReadableStream。

// 流式摘要 const stream = summarizer.summarizeStreaming(text); let result = ''; for await (const chunk of stream) { result += chunk; console.log('当前结果:', result); }
PROPERTY summarizer.summarizing

返回当前是否正在进行摘要操作。

METHOD summarizer.destroy()

销毁摘要器实例，释放资源。

配置选项详解

type - 摘要类型

'tl;dr' 默认 生成简短的摘要，适合快速了解内容
'key-points' - 提取关键要点，以列表形式呈现
'teaser' - 生成预告式摘要，吸引读者继续阅读
'headline' - 生成标题，最简短的摘要形式

format - 输出格式

'plain-text' 默认 纯文本格式
'markdown' - Markdown 格式，支持列表等格式

length - 摘要长度

'short' - 简短摘要
'medium' 默认 中等长度摘要
'long' - 详细摘要

示例演示

交互式 API 演示

基础摘要示例

// 完整的基础示例 async function basicSummarize() { // 1. 检查 API 可用性 const capabilities = await ai.summarizer.capabilities(); if (capabilities.available === 'no') { throw new Error('Summarizer API 不可用'); } // 2. 创建摘要器 const summarizer = await ai.summarizer.create({ type: 'tl;dr', format: 'plain-text', length: 'medium' }); // 3. 生成摘要 const text = '这是一段需要摘要的长文本...'; const summary = await summarizer.summarize(text); // 4. 输出结果 console.log('原文:', text); console.log('摘要:', summary); // 5. 销毁实例 summarizer.destroy(); }
