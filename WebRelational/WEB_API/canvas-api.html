CanvasRenderingContext2D

canvas Property

Returns a reference to the HTMLCanvasElement.

fillRect(x, y, width, height) Method

Draws a filled rectangle.

strokeRect(x, y, width, height) Method

Draws a rectangle outline.

clearRect(x, y, width, height) Method

Clears a rectangular area.

roundRect(x, y, w, h, radii) Method

Adds a rounded rectangle to the current path.

Path Methods

beginPath() / closePath() Method

Starts/closes a new path.

moveTo(x, y) / lineTo(x, y) Method

Moves the pen/draws a line to the specified point.

arc(x, y, radius, startAngle, endAngle, counterclockwise) Method

Adds a circular arc to the path.

arcTo(x1, y1, x2, y2, radius) Method

Adds an arc using control points and radius.

quadraticCurveTo(cpx, cpy, x, y) Method

Adds a quadratic Bézier curve.

bezierCurveTo(cp1x, cp1y, cp2x, cp2y, x, y) Method

Adds a cubic Bézier curve.

ellipse(x, y, rx, ry, rotation, startAngle, endAngle, ccw) Method

Adds an ellipse to the path.

fill() / stroke() / clip() Method

Fills/strokes/clips the current path.

Text Methods

fillText(text, x, y, maxWidth?) Method

Draws filled text.

strokeText(text, x, y, maxWidth?) Method

Draws text outline.

measureText(text) Method

Returns TextMetrics object.

Transform Methods

translate(x, y) / rotate(angle) / scale(x, y) Method

Applies transformations to the context.

transform(a, b, c, d, e, f) Method

Multiplies the current transformation matrix.

setTransform(a, b, c, d, e, f) / resetTransform() Method

Sets/resets the transformation matrix.

save() / restore() Method

Saves/restores the canvas state.

Pixel Manipulation

createImageData(width, height) Method

Creates a blank ImageData object.

getImageData(x, y, w, h) Method

Returns an ImageData object for the specified area.

putImageData(imageData, dx, dy) Method

Paints ImageData onto the canvas.

drawImage(image, dx, dy, dw?, dh?, sx?, sy?, sw?, sh?) Method

Draws an image onto the canvas.

ImageData Interface

data Property

Uint8ClampedArray containing RGBA data.

width / height Property

Dimensions of the ImageData.

TextMetrics Interface

width Property

The width of the text.

actualBoundingBoxLeft/Right/Ascent/Descent Property

The bounding box measurements.

Path2D Interface

new Path2D(path?) / new Path2D(d) Constructor

Creates a new Path2D object.

addPath(path, transform?) Method

Adds a path to the current path.

CanvasGradient Interface

addColorStop(offset, color) Method
