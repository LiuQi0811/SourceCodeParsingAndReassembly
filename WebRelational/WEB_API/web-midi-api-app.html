API 文档

Web MIDI API 完整文档

核心方法

METHOD navigator.requestMIDIAccess(options)

请求 MIDI 访问权限，返回 Promise<MIDIAccess>。

MIDIAccess

属性/方法说明
inputsMIDIInputMap - 输入设备集合
outputsMIDIOutputMap - 输出设备集合
onstatechange设备状态变化事件
sysexEnabled是否启用系统专属消息

MIDIInput

属性/方法说明
id设备唯一标识
name设备名称
manufacturer制造商
state连接状态
onmidimessageMIDI 消息事件处理

MIDIOutput

方法说明
send(data, timestamp)发送 MIDI 消息
clear()清除待发送的消息

MIDI 消息格式

状态字节说明
0x80-0x8FNote Off (通道 1-16)
0x90-0x9FNote On (通道 1-16)
0xA0-0xAFPolyphonic Key Pressure
0xB0-0xBFControl Change
0xC0-0xCFProgram Change
0xD0-0xDFChannel Pressure
0xE0-0xEFPitch Bend

使用示例

// 请求 MIDI 访问 const access = await navigator.requestMIDIAccess(); // 获取输入设备 access.inputs.forEach(input => { input.onmidimessage = (event) => { const [status, note, velocity] = event.data; console.log('MIDI消息:', status, note, velocity); }; }); // 获取输出设备并发送消息 access.outputs.forEach(output => { // Note On: 通道1, 音符60 (C4), 力度100 output.send([0x90, 60, 100]); // 1秒后 Note Off setTimeout(() => { output.send([0x80, 60, 0]); }, 1000); }); // 监听设备状态变化 access.onstatechange = (event) => { console.log('设备状态变化:', event.port.name, event.port.state); };

浏览器支持

Web MIDI API 需要 HTTPS 环境，并在 Chrome、Edge、Opera 等浏览器中得到支持。
