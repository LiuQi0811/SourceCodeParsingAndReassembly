URL Fragment Text Directives API 介绍

URL Fragment Text Directives 是一种新的 Web API，允许通过 URL 片段来高亮和滚动到页面中的特定文本。

这个 API 的主要用途是让用户可以直接链接到网页中的特定文本片段，而不需要页面作者预先设置锚点。

文本片段功能支持多种匹配模式，包括简单文本匹配、范围匹配和带上下文的精确匹配。

当用户访问带有文本片段的 URL 时，浏览器会自动滚动到指定位置并高亮显示匹配的文本。

高亮效果通常是黄色的背景标记，类似于用户手动选择的文本效果。

这个功能对于分享网页中的特定内容非常有用，用户可以直接链接到想要分享的文本片段。

滚动到指定位置是自动完成的，浏览器会确保目标文本在视口中可见。

第一章节

这是第一章节的内容。URL Fragment Text Directives API 允许开发者创建指向页面特定文本的链接。

这种链接格式为：#:~:text=要匹配的文本

浏览器会自动滚动到匹配的文本位置并高亮显示。

第二章节

这是第二章节的内容。文本片段功能支持多种匹配模式。

这是重要内容，需要特别注意。

范围匹配可以高亮从起始文本到结束文本之间的所有内容。

第三章节

这是第三章节的内容。前缀和后缀可以帮助更精确地定位文本。

这在页面中有多个相同文本时特别有用。

总结

URL Fragment Text Directives API 是一个强大的工具，可以改善用户体验，让分享和定位网页内容变得更加容易。

通过这个 API，用户可以直接链接到网页中的任何文本片段，无需页面作者进行任何特殊处理。

API 文档

URL Fragment Text Directives API 完整文档

概述

API URL Fragment Text Directives

允许通过 URL 片段来高亮和滚动到页面中的特定文本。

URL 格式

FORMAT #:~:text=textStart[,textEnd]

文本片段指令的基本格式。

// 简单文本匹配 https://example.com#:~:text=Hello%20World // 范围匹配 https://example.com#:~:text=Start,End // 带上下文的匹配 https://://example.com#:~:text=prefix-,target,-suffix

FragmentDirective 接口

INTERFACE FragmentDirective

表示文档中的文本片段指令。

// 检查浏览器支持 if ('fragmentDirective' in document) { console.log('FragmentDirective API 支持'); } // 访问 fragmentDirective const directive = document.fragmentDirective;

document.fragmentDirective

PROPERTY document.fragmentDirective

返回当前文档的 FragmentDirective 对象。

// 获取当前文档的 fragment directive const fragmentDirective = document.fragmentDirective; // 检查是否存在 if (fragmentDirective) { console.log('当前文档有 fragment directive'); }

文本片段格式详解

格式 说明 示例
text=目标 简单文本匹配 #:~:text=Hello
text=起始,结束 范围匹配 #:~:text=Start,End
text=前缀-,目标 带前缀匹配 #:~:text=Chapter-,Hello
text=目标,-后缀 带后缀匹配 #:~:text=Hello,-World
text=前缀-,目标,-后缀 完整上下文匹配 #:~:text=A-,B,-C

多个文本片段

MULTI 多个文本片段用 & 连接

可以在一个 URL 中指定多个文本片段。

// 多个文本片段 https://example.com#:~:text=First&text=Second&text=Third

编码规则

ENCODING 特殊字符需要 URL 编码

文本片段中的特殊字符需要进行 URL 编码。

// 空格编码为 %20 #:~:text=Hello%20World // 逗号编码为 %2C #:~:text=Hello%2C%20World // 使用 encodeURIComponent const text = "Hello, World!"; const encoded = encodeURIComponent(text); const url = `#:~:text=${encoded}`;
