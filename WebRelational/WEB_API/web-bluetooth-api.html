Bluetooth Interface

navigator.bluetooth.requestDevice(options) Promise<BluetoothDevice>

Requests a Bluetooth device from the user. Shows a device picker dialog.

options.filters Array<BluetoothLEScanFilter> - Service filters
options.acceptAllDevices boolean - Accept any device
options.optionalServices Array<UUID> - Optional services to access
navigator.bluetooth.getAvailability() Promise<boolean>

Returns whether the browser supports Bluetooth and it is available.

navigator.bluetooth.referringDevice BluetoothDevice | null

Returns the device that referred the user to this page via Bluetooth advertisement.

BluetoothDevice Interface

device.id string

Unique identifier for the device within the origin.

device.name string | null

Human-readable name of the device.

device.gatt BluetoothRemoteGATTServer | null

GATT server for the device.

device.gatt.connected boolean

Whether the device is currently connected.

device.gatt.connect() Promise<BluetoothRemoteGATTServer>

Connects to the GATT server.

device.gatt.disconnect() void

Disconnects from the GATT server.

device.watchAdvertisements() Promise<void>

Starts watching for advertisement packets from this device.

BluetoothRemoteGATTServer Interface

server.getPrimaryService(uuid) Promise<BluetoothRemoteGATTService>

Returns a primary service by UUID.

server.getPrimaryServices() Promise<BluetoothRemoteGATTService[]>

Returns all primary services.

BluetoothRemoteGATTService Interface

service.uuid UUID

UUID of the service.

service.isPrimary boolean

Whether this is a primary service.

service.getCharacteristic(uuid) Promise<BluetoothRemoteGATTCharacteristic>

Returns a characteristic by UUID.

service.getCharacteristics() Promise<BluetoothRemoteGATTCharacteristic[]>

Returns all characteristics of this service.

BluetoothRemoteGATTCharacteristic Interface

characteristic.uuid UUID

UUID of the characteristic.

characteristic.properties BluetoothCharacteristicProperties

Properties of the characteristic.

characteristic.value DataView | null

Current value of the characteristic.

characteristic.readValue() Promise<DataView>

Reads the current value of the characteristic.

characteristic.writeValue(value) Promise<void>

Writes a value to the characteristic (with response).

characteristic.writeValueWithoutResponse(value) Promise<void>

Writes a value without waiting for response.

characteristic.writeValueWithResponse(value) Promise<void>

Writes a value and waits for response.

characteristic.startNotifications() Promise<BluetoothRemoteGATTCharacteristic>

Starts notifications for this characteristic.

characteristic.stopNotifications() Promise<BluetoothRemoteGATTCharacteristic>

Stops notifications for this characteristic.

characteristic.getDescriptor(uuid) Promise<BluetoothRemoteGATTDescriptor>

Returns a descriptor by UUID.

characteristic.getDescriptors() Promise<BluetoothRemoteGATTDescriptor[]>

Returns all descriptors of this characteristic.

BluetoothCharacteristicProperties Interface

properties.broadcast boolean

Whether the characteristic supports broadcast.

properties.read boolean

Whether the characteristic supports read.

properties.writeWithoutResponse boolean

Whether the characteristic supports write without response.

properties.write boolean

Whether the characteristic supports write with response.

properties.notify boolean

Whether the characteristic supports notifications.

properties.indicate boolean

Whether the characteristic supports indications.

properties.authenticatedSignedWrites boolean

Whether the characteristic supports signed writes.

properties.reliableWrite boolean

Whether reliable write is supported.

properties.writableAuxiliaries boolean

Whether writable auxiliaries are supported.

descriptor.uuid UUID

UUID of the descriptor.

descriptor.value DataView | null

Current value of the descriptor.

descriptor.readValue() Promise<DataView>

Reads the current value of the descriptor.

descriptor.writeValue(value) Promise<void>

Writes a value to the descriptor.
