ContactsManager Interface

navigator.contacts Property

Returns the ContactsManager instance for accessing the Contact Picker API.

contacts.select(properties, options) Method

Opens the contact picker UI and returns selected contacts. Properties is an array of strings like ['name', 'email', 'tel']. Options can include { multiple: boolean }.

contacts.getProperties() Method

Returns a Promise that resolves with an array of supported property names.

ContactInfo Interface

contact.name Property

Array of strings representing the contact's names.

contact.email Property

Array of strings representing the contact's email addresses.

contact.tel Property

Array of strings representing the contact's phone numbers.

contact.address Property

Array of ContactAddress objects representing the contact's addresses.

contact.icon Property

Array of Blob objects representing the contact's avatar images.

ContactAddress Interface

address.country Property

The country name.

address.region Property

The region/state/province.

address.city Property

The city name.

address.street Property

The street address.

address.postalCode Property

The postal/zip code.

address.addressLine Property

Array of address lines.

ContactProperty Enum

Usage Example

// Check if API is supported if ('contacts' in navigator && 'ContactsManager' in window) { const contacts = navigator.contacts; // Get supported properties const supported = await contacts.getProperties(); console.log('Supported:', supported); // Select contacts const props = ['name', 'email', 'tel']; const options = { multiple: true }; const selected = await contacts.select(props, options); // Process selected contacts selected.forEach(contact => { console.log('Name:', contact.name); console.log('Email:', contact.email); console.log('Phone:', contact.tel); }); }
