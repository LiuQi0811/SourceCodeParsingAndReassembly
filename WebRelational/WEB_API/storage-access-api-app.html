API 文档

Storage Access API 完整接口文档

核心 Storage Access API

ASYNC document.requestStorageAccess()

请求为嵌入的跨源内容访问第三方 Cookie 的权限。返回一个 Promise，如果授予访问权限则 resolve，否则 reject。

返回值

Promise<void> Promise 成功时 resolve，拒绝时 reject 并抛出异常
// 请求存储访问权限 try { await document.requestStorageAccess(); console.log('存储访问权限已授予'); // 现在可以访问第三方 Cookie } catch (error) { console.error('存储访问权限被拒绝:', error); // 处理权限被拒绝的情况 }
ASYNC document.hasStorageAccess()

检查当前文档是否已获得存储访问权限。返回一个 Promise，解析为布尔值。

返回值

Promise<boolean> Promise true 表示已有访问权限，false 表示没有
// 检查是否已有存储访问权限 const hasAccess = await document.hasStorageAccess(); if (hasAccess) { console.log('已有存储访问权限'); } else { console.log('没有存储访问权限，需要请求'); }
PROPERTY Document.requestStorageAccess

检查浏览器是否支持 Storage Access API。

// 检查 API 支持 if (document.requestStorageAccess) { console.log('浏览器支持 Storage Access API'); } else { console.log('浏览器不支持 Storage Access API'); }

权限查询 API

ASYNC navigator.permissions.query({ name: 'storage-access' })

查询存储访问权限的状态。

参数

name string 权限名称，固定为 'storage-access'

返回状态

granted string 权限已授予
denied string 权限已拒绝
prompt string 需要用户确认
// 查询存储访问权限状态 const permission = await navigator.permissions.query({ name: 'storage-access' }); console.log('权限状态:', permission.state); // 可能的值: 'granted', 'denied', 'prompt' // 监听权限变化 permission.onchange = () => { console.log('权限状态变更为:', permission.state); };
ASYNC navigator.permissions.query({ name: 'persistent-storage' })

查询持久化存储权限状态。

// 请求持久化存储 if (navigator.storage && navigator.storage.persist) { const isPersisted = await navigator.storage.persist(); console.log('持久化存储:', isPersisted ? '已启用' : '未启用'); } // 检查持久化状态 if (navigator.storage && navigator.storage.persisted) { const persisted = await navigator.storage.persisted(); console.log('持久化状态:', persisted); }

文件系统句柄 API

ASYNC window.showOpenFilePicker(options?)

显示文件选择器，返回文件句柄数组。

Options 参数

multiple boolean 是否允许多选，默认 false
excludeAcceptAllOption boolean 是否排除"所有文件"选项
types array 可接受的文件类型
// 打开文件选择器 const [fileHandle] = await window.showOpenFilePicker({ multiple: false, types: [{ description: '文本文件', accept: { 'text/plain': ['.txt'] } }] }); // 获取文件内容 const file = await fileHandle.getFile(); const contents = await file.text(); console.log('文件内容:', contents);
ASYNC window.showSaveFilePicker(options?)

显示保存文件对话框，返回文件句柄。

// 保存文件 const handle = await window.showSaveFilePicker({ suggestedName: 'untitled.txt', types: [{ description: '文本文件', accept: { 'text/plain': ['.txt'] } }] }); // 写入内容 const writable = await handle.createWritable(); await writable.write('Hello, World!'); await writable.close();
ASYNC window.showDirectoryPicker(options?)

显示目录选择器，返回目录句柄。

// 选择目录 const dirHandle = await window.showDirectoryPicker(); // 遍历目录 for await (const entry of dirHandle.values()) { console.log(entry.kind, entry.name); } // 获取文件 const fileHandle = await dirHandle.getFileHandle('example.txt'); // 创建新文件 const newFileHandle = await dirHandle.getFileHandle('new.txt', { create: true });
ASYNC handle.requestPermission(options?)

请求文件/目录句柄的读写权限。

// 请求读写权限 const permission = await handle.requestPermission({ mode: 'readwrite' // 或 'read' }); if (permission === 'granted') { console.log('权限已授予'); }
ASYNC handle.queryPermission(options?)

查询文件/目录句柄的权限状态。

// 查询权限状态 const permission = await handle.queryPermission({ mode: 'readwrite' }); // 可能的值: 'granted', 'denied', 'prompt' if (permission === 'prompt') { // 需要请求权限 await handle.requestPermission({ mode: 'readwrite' }); }

存储事件 API

EVENT window.onstorage

监听 localStorage 变化事件（跨标签页同步）。

// 监听存储变化 window.addEventListener('storage', (event) => { console.log('Key:', event.key); console.log('Old Value:', event.oldValue); console.log('New Value:', event.newValue); console.log('URL:', event.url); console.log('Storage Area:', event.storageArea); });
EVENT navigator.storage.onestimatechange

存储配额估算变化事件。

// 获取存储估算 const estimate = await navigator.storage.estimate(); console.log('配额:', estimate.quota); console.log('已用:', estimate.usage); // 监听变化 navigator.storage.onestimatechange = () => { console.log('存储估算已更新'); };
ASYNC navigator.storage.estimate()

获取存储配额和使用量估算。

// 获取存储估算 const estimate = await navigator.storage.estimate(); const usedMB = (estimate.usage / 1024 / 1024).toFixed(2); const quotaMB = (estimate.quota / 1024 / 1024).toFixed(2); console.log(`已使用: ${usedMB} MB`); console.log(`总配额: ${quotaMB} MB`); console.log(`使用率: ${(estimate.usage / estimate.quota * 100).toFixed(1)}%`);
