高熵值

获取更详细的用户代理信息

什么是高熵值？

高熵值提示需要用户授权才能获取，包含更详细的设备信息，如架构、位数、完整版本等。

对比传统UA

比较 User-Agent Client Hints 与传统 User-Agent 字符串

为什么要使用 Client Hints？

传统的 User-Agent 字符串包含过多信息，容易被用于指纹追踪。Client Hints 提供了更隐私友好的方式，让网站只获取必要的信息。

API 文档

User-Agent Client Hints API 完整文档

概述

API navigator.userAgentData

返回 NavigatorUAData 对象，包含用户代理信息。

NavigatorUAData 属性

属性 类型 说明
brandsUADataValues[]浏览器品牌和版本
mobileboolean是否为移动设备
platformstring操作系统平台

NavigatorUAData 方法

METHOD getHighEntropyValues(hints)

返回 Promise，解析为包含高熵值提示的对象。

// 获取高熵值 const uaData = navigator.userAgentData; const highEntropy = await uaData.getHighEntropyValues([ 'architecture', 'bitness', 'model', 'platformVersion', 'uaFullVersion', 'fullVersionList' ]); console.log(highEntropy);

高熵值提示

提示名 说明
architectureCPU 架构 (x86, arm)
bitness系统位数 (64, 32)
model设备型号
platformVersion操作系统版本
uaFullVersion浏览器完整版本
fullVersionList完整品牌版本列表

使用示例

// 检测 API 支持 if ('userAgentData' in navigator) { const ua = navigator.userAgentData; // 获取基础信息 console.log('品牌:', ua.brands); console.log('平台:', ua.platform); console.log('移动设备:', ua.mobile); // 获取高熵值 const highEntropy = await ua.getHighEntropyValues([ 'architecture', 'bitness', 'platformVersion' ]); console.log('架构:', highEntropy.architecture); console.log('位数:', highEntropy.bitness); console.log('系统版本:', highEntropy.platformVersion); }
