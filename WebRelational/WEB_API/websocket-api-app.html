API 文档

WebSocket API 完整文档

核心接口

CONSTRUCTOR new WebSocket(url, protocols)

创建一个新的 WebSocket 连接。

PROPERTY WebSocket.readyState

连接状态: CONNECTING(0), OPEN(1), CLOSING(2), CLOSED(3)

PROPERTY WebSocket.url

WebSocket 连接的 URL。

PROPERTY WebSocket.protocol

服务器选择的子协议。

PROPERTY WebSocket.binaryType

二进制数据类型: "blob" 或 "arraybuffer"

PROPERTY WebSocket.bufferedAmount

排队等待发送的数据字节数。

方法

方法说明
WebSocket.send(data)发送数据（文本或二进制）
WebSocket.close(code, reason)关闭连接

事件

事件说明
onopen连接建立时触发
onmessage收到消息时触发
onerror发生错误时触发
onclose连接关闭时触发

关闭代码

代码名称说明
1000CLOSE_NORMAL正常关闭
1001CLOSE_GOING_AWAY端点离开
1002CLOSE_PROTOCOL_ERROR协议错误
1003CLOSE_UNSUPPORTED不支持的数据类型
1006CLOSE_ABNORMAL异常关闭
1007Unsupported Data数据类型不一致
1008Policy Violation策略违规
1009CLOSE_TOO_LARGE消息过大
1010Missing Extension缺少扩展
1011Internal Error内部错误
1012Service Restart服务重启
1013Try Again Later稍后重试
