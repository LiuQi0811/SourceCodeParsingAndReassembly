API 文档

Window Controls Overlay API 完整文档

核心接口

INTERFACE WindowControlsOverlay

窗口控制覆盖层接口，通过 navigator.windowControlsOverlay 访问。

PROPERTY visible

布尔值，表示窗口控制覆盖层是否可见。

METHOD getBoundingClientRect()

返回窗口控制按钮区域的 DOMRect。

EVENT geometrychange

当覆盖层几何信息变化时触发。

CSS 环境变量

变量说明
env(titlebar-area-height)标题栏区域高度
env(titlebar-area-width)标题栏区域宽度
env(titlebar-area-x)标题栏区域 X 坐标
env(titlebar-area-y)标题栏区域 Y 坐标

CSS 属性

属性说明
app-region: drag元素可拖动窗口
app-region: no-drag元素不可拖动

浏览器支持

最低版本要求：
  • Chrome 105+
  • Edge 105+
  • Opera 91+
  • Firefox: 不支持
  • Safari: 不支持
