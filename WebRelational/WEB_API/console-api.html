Output Methods

console.log(data, ...args) Method

Outputs a message to the console. Supports format specifiers.

console.info(data, ...args) Method

Outputs an informational message. Similar to log but may have different styling.

console.warn(data, ...args) Method

Outputs a warning message. Usually displayed with yellow styling.

console.error(data, ...args) Method

Outputs an error message. Usually displayed with red styling.

console.debug(data, ...args) Method

Outputs a debug message. May be filtered by log level.

Timer Methods

console.time(label) Method

Starts a timer with the given label.

console.timeLog(label, ...data) Method

Logs the current value of a timer.

console.timeEnd(label) Method

Stops the timer and logs the elapsed time.

Counter Methods

console.count(label) Method

Logs the number of times count() has been called with the given label.

console.countReset(label) Method

Resets the counter for the given label.

Grouping Methods

console.group(label) Method

Creates a new inline group, indenting subsequent messages.

console.groupCollapsed(label) Method

Creates a new inline group that is collapsed by default.

console.groupEnd() Method

Exits the current inline group.

Other Methods

console.table(data, columns?) Method

Displays data in tabular format.

console.dir(object, options?) Method

Displays an interactive listing of object properties.

console.dirxml(object) Method

Displays an XML/HTML element representation if possible.

console.assert(condition, ...data) Method

Logs an error if condition is false.

console.trace(label?) Method

Outputs a stack trace.

console.clear() Method

Clears the console.

Format Specifiers

%s Specifier

Formats as string.

%d or %i Specifier

Formats as integer.

%f Specifier

Formats as floating point.

%o or %O Specifier

Formats as object.

%c Specifier

Applies CSS styling to output.
