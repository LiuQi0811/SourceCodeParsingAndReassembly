API 文档

Virtual Keyboard API 完整文档

概述

API navigator.virtualKeyboard

Virtual Keyboard API 允许网页控制虚拟键盘的显示和获取键盘的几何信息。

属性

属性类型说明
boundingRectDOMRect虚拟键盘的边界矩形

方法

方法说明
show()显示虚拟键盘
hide()隐藏虚拟键盘

事件

事件说明
geometrychange键盘几何信息变化时触发

CSS 属性

属性值说明
virtual-keyboard-policyauto | manual控制键盘行为模式

使用示例

// 检测支持 if ('virtualKeyboard' in navigator) { const vk = navigator.virtualKeyboard; // 显示键盘 vk.show(); // 隐藏键盘 vk.hide(); // 获取边界 const rect = vk.boundingRect; console.log('键盘高度:', rect.height); // 监听变化 vk.addEventListener('geometrychange', (e) => { console.log('键盘几何变化'); }); } // CSS 策略 .editable { virtual-keyboard-policy: manual; }

浏览器支持

实验性 API - Virtual Keyboard API 目前处于实验阶段，主要在触屏设备上使用。
