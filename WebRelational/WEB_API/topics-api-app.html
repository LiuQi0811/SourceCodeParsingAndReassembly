Topics API 完整接口文档

Topics API 概述

Topics API 是一个隐私保护 API，允许网站根据用户的浏览历史推断用户感兴趣的主题，同时保护用户隐私。主题在本地计算，不涉及跨站跟踪。

PROPERTY document.browsingTopics

检查 Topics API 是否可用。

// 检查 API 可用性 if ('browsingTopics' in document) { console.log('Topics API 可用'); }

核心特性

隐私保护 特性 主题在本地计算，不发送浏览历史
主题分类 特性 使用标准化分类体系（350+ 分类）
时间窗口 特性 基于最近一周的浏览历史
主题数量 特性 每周最多返回 3 个主题

API 方法

ASYNC document.browsingTopics(options?)

获取用户感兴趣的主题列表。

Options 参数

skipObservation boolean 是否跳过观察记录，默认 false

返回值

topic string 主题名称
taxonomyVersion string 分类版本
modelVersion string 模型版本
configVersion string 配置版本
// 获取主题 const topics = await document.browsingTopics(); // 返回示例 [ { topic: "Technology & Computing", taxonomyVersion: "1", modelVersion: "1", configVersion: "1" } ]
ASYNC document.browsingTopics({ skipObservation: true })

获取主题但不记录本次调用为观察。

// 获取主题但不记录观察 const topics = await document.browsingTopics({ skipObservation: true });

HTTP 头

REQUEST Sec-Browsing-Topics

在 fetch 请求中自动附加主题信息。

// fetch 请求会自动附加主题头 fetch('https://ads.example.com/bid', { mode: 'cors', credentials: 'include' }); // 请求头示例 // Sec-Browsing-Topics: (101);p=0.85, (205);p=0.72
RESPONSE Observe-Browsing-Topics

响应头，指示是否观察当前页面的主题。

// 服务器响应头 Observe-Browsing-Topics: ?1 // 启用观察 Observe-Browsing-Topics: ?0 // 禁用观察
RESPONSE Permissions-Policy

控制哪些源可以访问 Topics API。

// 允许特定源访问 Permissions-Policy: browsing-topics=(self "https://trusted.com") // 完全禁用 Permissions-Policy: browsing-topics=()

权限策略

browsing-topics 权限

默认值 * 默认允许所有同源访问
允许值 list self, 或指定的源列表
// HTML iframe 配置 <iframe allow="browsing-topics" src="..."></iframe> // HTTP 响应头配置 Permissions-Policy: browsing-topics=(self "https://partner.com") // 禁用 Permissions-Policy: browsing-topics=()

相关权限

join-ad-interest-group 权限 允许加入广告兴趣组
run-ad-auction 权限 允许运行广告拍卖
attribution-reporting 权限 允许归因报告

示例演示

交互式 API 演示

基础用法示例

// 检查 API 支持 if (!('browsingTopics' in document)) { console.log('浏览器不支持 Topics API'); return; } // 获取用户主题 try { const topics = await document.browsingTopics(); console.log('用户感兴趣的主题:'); topics.forEach((topic, index) => { console.log(`${index + 1}. ${topic.topic}`); }); } catch (error) { console.error('获取主题失败:', error); }
// 点击运行查看结果

Fetch 请求示例

// 带主题的 fetch 请求 fetch('https://ads.example.com/bid', { method: 'POST', mode: 'cors', credentials: 'include', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ adSlot: 'banner-1' }) }) .then(response => response.json()) .then(data => { console.log('广告响应:', data); }); // 请求会自动包含 Sec-Browsing-Topics 头
// 点击运行查看结果

iframe 嵌入示例

<!-- 启用 Topics API 的 iframe --> <iframe src="https://partner.example.com/ad-frame" allow="browsing-topics; join-ad-interest-group; run-ad-auction" width="300" height="250"> </iframe> <!-- 完整的广告技术权限 --> <iframe src="https://dsp.example.com" allow="browsing-topics; join-ad-interest-group; run-ad-auction; attribution-reporting"> </iframe>
