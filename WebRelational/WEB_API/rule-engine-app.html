规则管理接口

GET /api/v1/rules

获取规则列表，支持分页、筛选和排序

查询参数

page integer 页码，默认1
pageSize integer 每页数量，默认20
groupId string 分组ID筛选
status string 状态筛选: active/inactive
keyword string 关键词搜索
{ "code": 200, "data": { "list": [...], "total": 100, "page": 1, "pageSize": 20 } }
GET /api/v1/rules/{id}

获取单个规则详情

路径参数

id string 规则ID
POST /api/v1/rules

创建新规则

请求体

{ "name": "风控规则001", "code": "RISK_001", "groupId": "group_001", "priority": 100, "description": "高风险交易拦截规则", "expression": { "conditions": [ {"field": "amount", "operator": "gt", "value": 50000} ], "logic": "and" }, "actions": [ {"type": "reject", "message": "交易金额超限"} ], "status": "active" }
PUT /api/v1/rules/{id}

更新规则信息

DELETE /api/v1/rules/{id}

删除规则（软删除）

POST /api/v1/rules/{id}/enable

启用规则

POST /api/v1/rules/{id}/disable

禁用规则

POST /api/v1/rules/batch

批量操作规则

{ "ruleIds": ["rule_001", "rule_002"], "action": "enable" // enable/disable/delete }

分组管理接口

GET /api/v1/groups

获取分组列表

GET /api/v1/groups/{id}

获取分组详情

POST /api/v1/groups

创建新分组

{ "name": "风控规则组", "code": "RISK_CONTROL", "description": "用于风险控制的规则集合" }
PUT /api/v1/groups/{id}

更新分组信息

DELETE /api/v1/groups/{id}

删除分组（分组下有规则时不可删除）

GET /api/v1/groups/{id}/rules

获取分组下的所有规则

规则执行接口

POST /api/v1/execute

执行单个规则

{ "ruleId": "rule_001", "params": { "amount": 100000, "userId": "user_123", "transactionId": "txn_456" }, "options": { "debug": true, "timeout": 5000 } }
// 响应示例 { "code": 200, "data": { "ruleId": "rule_001", "result": "reject", "message": "交易金额超限", "actions": [...], "executionTime": 12, "debugInfo": {...} } }
POST /api/v1/execute/batch

批量执行规则

{ "ruleIds": ["rule_001", "rule_002"], "params": {...}, "stopOnFirstMatch": true }
POST /api/v1/execute/group/{groupId}

执行分组下所有规则

POST /api/v1/execute/chain

执行规则链

{ "chainId": "risk_chain", "params": {...}, "passOutput": true }
POST /api/v1/test

测试规则（不保存执行记录）

版本管理接口

GET /api/v1/rules/{id}/versions

获取规则版本历史

GET /api/v1/rules/{id}/versions/{versionId}

获取特定版本详情

POST /api/v1/rules/{id}/versions

创建新版本（手动保存快照）

POST /api/v1/rules/{id}/rollback/{versionId}

回滚到指定版本

POST /api/v1/rules/{id}/compare

对比两个版本

{ "versionA": "v1.0.0", "versionB": "v1.1.0" }

导入导出接口

GET /api/v1/export

导出规则

查询参数

scope string 导出范围: all/group/selected
groupId string 分组ID（scope=group时必填）
ruleIds array 规则ID列表（scope=selected时必填）
format string 导出格式: json/yaml/xml
includeVersions boolean 是否包含版本历史
POST /api/v1/import

导入规则

// Content-Type: multipart/form-data { "file": "<binary>", "strategy": "skip", // skip/overwrite/rename "validateOnly": false }
POST /api/v1/import/preview

预览导入结果（不实际导入）
