API 文档

Web Authentication API 完整文档

核心方法

CREATE navigator.credentials.create(options)

创建新的凭证，用于注册流程。

GET navigator.credentials.get(options)

获取凭证，用于认证流程。

PublicKeyCredentialCreationOptions

属性类型说明
rpobject依赖方信息 (id, name)
userobject用户信息 (id, name, displayName)
challengeBufferSource随机挑战值
pubKeyCredParamsarray支持的算法
authenticatorSelectionobject认证器选择条件
timeoutnumber超时时间 (ms)
attestationstring证明方式

PublicKeyCredentialRequestOptions

属性类型说明
challengeBufferSource随机挑战值
rpIdstring依赖方 ID
allowCredentialsarray允许的凭证列表
userVerificationstring用户验证要求
timeoutnumber超时时间 (ms)

使用示例

// 注册 const credential = await navigator.credentials.create({ publicKey: { rp: { name: "My App", id: window.location.hostname }, user: { id: new Uint8Array(16), name: "user@example.com", displayName: "User" }, challenge: new Uint8Array(32), pubKeyCredParams: [ { type: "public-key", alg: -7 }, // ES256 { type: "public-key", alg: -257 } // RS256 ], authenticatorSelection: { authenticatorAttachment: "platform", userVerification: "required" } } }); // 认证 const assertion = await navigator.credentials.get({ publicKey: { challenge: new Uint8Array(32), rpId: window.location.hostname, userVerification: "required" } });

浏览器支持

HTTPS 要求： WebAuthn 仅在 HTTPS 或 localhost 环境下可用。
