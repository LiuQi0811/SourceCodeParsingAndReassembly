核心方法

                    

                        METHOD
                        element.animate(keyframes, options)
                        
在元素上创建并运行动画，返回 Animation 对象。

                    


                    
Animation 对象

                    

                        
                            
属性/方法类型说明

                        
                        
                            
playStatestring播放状态 (idle/running/paused/finished)

                            
currentTimenumber当前时间

                            
startTimenumber开始时间

                            
playbackRatenumber播放速率

                            
effectAnimationEffect动画效果

                            
timelineAnimationTimeline时间轴

                            
finishedPromise动画完成 Promise

                            
readyPromise动画就绪 Promise

                            
play()方法播放动画

                            
pause()方法暂停动画

                            
reverse()方法反向播放

                            
finish()方法完成动画

                            
cancel()方法取消动画

                        
                    


                    
KeyframeEffect

                    

                        
属性/方法说明

                        
                            
new KeyframeEffect(target, keyframes, options)创建关键帧效果

                            
getKeyframes()获取关键帧数组

                            
setKeyframes(keyframes)设置关键帧

                            
target目标元素

                        
                    


                    
动画选项

                    

                        
选项类型说明

                        
                            
durationnumber持续时间 (ms)

                            
delaynumber延迟时间 (ms)

                            
iterationsnumber迭代次数 (Infinity 无限)

                            
easingstring缓动函数

                            
fillstring填充模式 (none/forwards/backwards/both)

                            
directionstring方向 (normal/reverse/alternate/alternate-reverse)

                        
                    


                    
使用示例

                    

                        // 基本动画
const animation = element.animate([
    { opacity: 0, transform: 'translateX(-100px)' },
    { opacity: 1, transform: 'translateX(0)' }
], {
    duration: 1000,
    easing: 'ease-out',
    fill: 'forwards'
});

// 控制动画
animation.pause();
animation.play();
animation.reverse();

// 监听完成
animation.finished.then(() => {
    console.log('动画完成');
});

// 使用 KeyframeEffect
const effect = new KeyframeEffect(
    element,
    [{ transform: 'rotate(0deg)' }, { transform: 'rotate(360deg)' }],
    { duration: 1000, iterations: Infinity }
);
const anim = new Animation(effect, document.timeline);
anim.play();
