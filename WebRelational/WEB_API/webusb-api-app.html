API 文档

WebUSB API 完整文档

核心接口

INTERFACE USB

WebUSB API 的主入口，通过 navigator.usb 访问。

INTERFACE USBDevice

代表一个 USB 设备。

INTERFACE USBConfiguration

USB 配置描述符。

INTERFACE USBInterface

USB 接口描述符。

INTERFACE USBEndpoint

USB 端点描述符。

主要方法

方法说明
navigator.usb.requestDevice(filters)请求 USB 设备
navigator.usb.getDevices()获取已授权设备
device.open()打开设备
device.close()关闭设备
device.selectConfiguration(configValue)选择配置
device.claimInterface(interfaceNumber)声明接口
device.releaseInterface(interfaceNumber)释放接口
device.controlTransferIn(setup, length)控制输入
device.controlTransferOut(setup, data)控制输出
device.transferIn(endpointNumber, length)批量输入
device.transferOut(endpointNumber, data)批量输出
