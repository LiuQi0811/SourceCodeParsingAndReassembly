API 状态 说明

⚡ 快速开始

// 创建编码器 const encoder = new TextEncoder(); const bytes = encoder.encode('Hello 世界'); console.log(bytes); // Uint8Array [72, 101, 108, 108, 111, 32, 228, 184, 150, 231, 149, 140] // 创建解码器 const decoder = new TextDecoder('utf-8'); const text = decoder.decode(bytes); console.log(text); // "Hello 世界" // 编码到现有缓冲区 const buffer = new Uint8Array(20); const result = encoder.encodeInto('Hello', buffer); console.log(result.read, result.written);

TextEncoder 接口

TextEncoder 将字符串编码为 UTF-8 字节数组

📋 属性

属性 类型 值 说明
encoding string "utf-8" 编码名称（始终为 utf-8）

🔧 方法

方法 参数 返回值 说明
encode() string? Uint8Array 将字符串编码为字节数组
encodeInto() string, Uint8Array object 编码到现有缓冲区

📦 encodeInto 返回值

属性 类型 说明
read number 已读取的 UTF-16 代码单元数
written number 已写入的 UTF-8 字节数

TextDecoder 接口

TextDecoder 将字节数组解码为字符串

🔧 构造函数

new TextDecoder(encoding?, options?)
参数 类型 默认值 说明
encoding string "utf-8" 编码名称
options.fatal boolean false 解码失败时是否抛出异常
options.ignoreBOM boolean false 是否忽略 BOM 标记

📋 属性

属性 类型 说明
encoding string 编码名称
fatal boolean 是否为致命模式
ignoreBOM boolean 是否忽略 BOM

🔧 方法

方法 参数 返回值 说明
decode() buffer?, options? string 将字节数组解码为字符串

⚙️ decode Options

选项 类型 默认值 说明
stream boolean false 是否为流式解码
