Animation Demo

Registered custom properties can be animated! This is one of the main benefits of the CSS Properties and Values API.

Animation CSS
/* Registered property enables animation */ @property --animation-color { syntax: ''; inherits: false; initial-value: #f97316; } @keyframes colorChange { from { --animation-color: #f97316; } to { --animation-color: #3b82f6; } } .animated-element { animation: colorChange 3s ease-in-out infinite alternate; background: var(--animation-color); }

CSS.registerProperty()

CSS.registerProperty(definition) Method

Registers a custom CSS property. Must be called before the property is used.

CSS.registerProperty({ name: '--my-color', syntax: '', inherits: false, initialValue: '#f97316' });

PropertyDefinition Interface

name Property (required)

The name of the custom property. Must start with -- (double dash).

syntax Property (required)

A string defining the allowed syntax for the property.

inherits Property (required)

Whether the property should inherit by default.

initialValue Property (optional)

The initial value of the property. Required if syntax is not '*'

@property CSS Rule

@property CSS At-Rule

CSS at-rule for registering custom properties directly in CSS.

@property --my-color { syntax: ''; inherits: false; initial-value: #f97316; }

CSSPropertyRule Interface

rule.name Property (readonly)

Returns the name of the custom property.

rule.syntax Property (readonly)

Returns the syntax string of the property.

rule.inherits Property (readonly)

Returns whether the property inherits.

rule.initialValue Property (readonly)

Returns the initial value of the property.
