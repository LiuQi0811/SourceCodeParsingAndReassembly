Window Interface

window.innerWidth / innerHeight Property

Returns the interior width/height of the window in pixels.

window.outerWidth / outerHeight Property

Returns the width/height of the outside of the browser window.

window.scrollX / scrollY Property

Returns the number of pixels the document has been scrolled.

window.scrollTo(x, y) / scrollTo(options) Method

Scrolls to a particular set of coordinates in the document.

window.scrollBy(x, y) / scrollBy(options) Method

Scrolls the document by the given amount.

Element Interface

element.getBoundingClientRect() Method

Returns a DOMRect providing information about the size and position relative to the viewport.

element.getClientRects() Method

Returns a collection of DOMRect objects for each border box.

element.scrollIntoView(options) Method

Scrolls the element's ancestor containers so the element is visible.

offsetWidth / offsetHeight / offsetTop / offsetLeft Properties

Returns the layout size and position of the element.

clientWidth / clientHeight / clientTop / clientLeft Properties

Returns the inner size and border widths of the element.

scrollWidth / scrollHeight / scrollTop / scrollLeft Properties

Returns the scrolling size and scroll position of the element.

MouseEvent Interface

clientX / clientY Property

Coordinates within the viewport.

pageX / pageY Property

Coordinates within the entire document.

screenX / screenY Property

Coordinates within the screen.

offsetX / offsetY Property

Coordinates within the target element.

movementX / movementY Property

Delta since the last mousemove event.

Screen Interface

screen.width / height Property

The screen's dimensions in pixels.

screen.availWidth / availHeight Property

The available screen dimensions.

screen.colorDepth / pixelDepth Property

The color depth of the screen.

screen.orientation Property

Returns the ScreenOrientation object with type and angle.

VisualViewport Interface

visualViewport.width / height Property

The visual viewport dimensions.

visualViewport.offsetLeft / offsetTop Property

Offset from the layout viewport.

visualViewport.scale Property

The pinch-zoom scaling factor.
