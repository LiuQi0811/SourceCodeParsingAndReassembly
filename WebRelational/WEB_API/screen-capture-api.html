MediaRecorder API 示例 
// 创建 MediaRecorder
const stream = await navigator.mediaDevices.getDisplayMedia();
const recorder = new MediaRecorder(stream, {
    mimeType: 'video/webm;codecs=vp9',
    videoBitsPerSecond: 2500000
});

const chunks = [];
recorder.ondataavailable = e => chunks.push(e.data);

recorder.onstop = () => {
    const blob = new Blob(chunks, { type: 'video/webm' });
    const url = URL.createObjectURL(blob);
    // 下载或播放录制文件
};

recorder.start(1000); // 每1秒触发一次 ondataavailable

画中画选项

画中画 API 示例 
// 进入画中画模式
const video = document.querySelector('video');
await video.requestPictureInPicture();

// 退出画中画模式
await document.exitPictureInPicture();

// 监听画中画事件
video.addEventListener('enterpictureinpicture', () => {
    console.log('进入画中画模式');
});

video.addEventListener('leavepictureinpicture', () => {
    console.log('退出画中画模式');
});

// 获取画中画窗口
const pipWindow = document.pictureInPictureElement;
if (pipWindow) {
    console.log('宽度:', pipWindow.videoWidth);
    console.log('高度:', pipWindow.videoHeight);
}

API 参考

核心方法

navigator.mediaDevices.getDisplayMedia(constraints)
请求用户选择并捕获屏幕、窗口或标签页。返回 Promise<MediaStream>
参数: constraints - MediaStreamConstraints 对象
const stream = await navigator.mediaDevices.getDisplayMedia({ video: true, audio: true });

视频约束属性

displaySurface
指定要捕获的显示表面类型
可选值: "monitor" | "window" | "browser" | "application"
logicalSurface
是否捕获逻辑表面而非物理表面
类型: boolean
cursor
控制光标在捕获中的显示方式
可选值: "always" | "motion" | "never"
width, height
视频的宽度和高度约束
类型: ConstrainULong (数值或范围对象)
frameRate
视频帧率约束
类型: ConstrainDouble (数值或范围对象)
aspectRatio
视频宽高比约束
类型: ConstrainDouble
resizeMode
视频缩放模式
可选值: "none" | "crop-and-scale"

音频约束属性

suppressLocalAudioPlayback
是否在捕获时抑制本地音频播放
类型: boolean
echoCancellation
是否启用回声消除
类型: boolean
noiseSuppression
是否启用噪声抑制
类型: boolean

高级约束选项

preferCurrentTab
是否优先选择当前标签页
类型: boolean
selfBrowserSurface
控制是否允许选择自身浏览器表面
可选值: "include" | "exclude"
systemAudio
控制是否包含系统音频
可选值: "include" | "exclude"
surfaceSwitching
控制是否允许在捕获过程中切换表面
可选值: "include" | "exclude"
monitorTypeSurfaces
控制是否包含显示器类型表面
可选值: "include" | "exclude"

MediaStreamTrack 方法

track.getCapabilities()
返回轨道支持的所有能力和约束范围
返回: MediaTrackCapabilities 对象
track.getConstraints()
返回当前应用的约束
返回: MediaTrackConstraints 对象
track.getSettings()
返回当前生效的设置值
返回: MediaTrackSettings 对象
track.applyConstraints(constraints)
应用新的约束到轨道
参数: constraints - MediaTrackConstraints 对象
await track.applyConstraints({ width: { ideal: 1920 }, height: { ideal: 1080 }, frameRate: { ideal: 30 } });
track.stop()
停止轨道，释放相关资源
无参数，无返回值
track.clone()
克隆轨道，返回新的 MediaStreamTrack
返回: MediaStreamTrack

MediaStreamTrack 事件

ended
当轨道永久停止时触发
触发场景: 用户停止共享、调用 stop() 方法等
mute
当轨道变为静音状态时触发
track.muted 变为 true
unmute
当轨道取消静音状态时触发
track.muted 变为 false

MediaStream 属性和方法

stream.getTracks()
返回流中的所有轨道
返回: MediaStreamTrack[]
stream.getVideoTracks()
返回流中的所有视频轨道
返回: MediaStreamTrack[]
stream.getAudioTracks()
返回流中的所有音频轨道
返回: MediaStreamTrack[]
stream.addTrack(track)
向流中添加轨道
参数: track - MediaStreamTrack
stream.removeTrack(track)
从流中移除轨道
参数: track - MediaStreamTrack

Picture-in-Picture API

video.requestPictureInPicture()
请求进入画中画模式
返回: Promise<PictureInPictureWindow>
document.exitPictureInPicture()
退出画中画模式
返回: Promise<void>
document.pictureInPictureElement
返回当前处于画中画模式的元素
返回: HTMLVideoElement | null
document.pictureInPictureEnabled
检查画中画功能是否可用
返回: boolean

MediaRecorder API

new MediaRecorder(stream, options)
创建媒体录制器
参数: stream - MediaStream, options - { mimeType, videoBitsPerSecond, audioBitsPerSecond }
recorder.start(timeslice)
开始录制
参数: timeslice - 可选，分片间隔(毫秒)
recorder.stop()
停止录制，触发 dataavailable 和 stop 事件
无参数
recorder.pause()
暂停录制
无参数
recorder.resume()
恢复录制
无参数
recorder.requestData()
请求立即触发 dataavailable 事件
无参数
MediaRecorder.isTypeSupported(mimeType)
检查 MIME 类型是否支持
参数: mimeType - 字符串，返回: boolean
