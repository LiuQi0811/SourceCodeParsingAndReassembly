Service Worker Code
// Service Worker for Content Index API self.addEventListener('install', (event) => { self.skipWaiting(); }); self.addEventListener('activate', (event) => { event.waitUntil(clients.claim()); }); // Handle contentdelete event self.addEventListener('contentdelete', (event) => { console.log('[SW] Content deleted:', event.id); // Clean up cached content event.waitUntil( caches.open('content-cache').then((cache) => { return cache.delete(new Request(event.id)); }) ); }); // Access Content Index const index = self.index; // Add content async function addContent(data) { await index.add({ id: data.id, title: data.title, description: data.description, url: data.url, category: data.category, icons: data.icons }); } // Delete content async function deleteContent(id) { await index.delete(id); } // Get all content async function getAllContent() { return await index.getAll(); }

ContentIndex Interface

serviceWorkerRegistration.index Property

Returns the ContentIndex interface for managing indexed content.

index.add(contentDescription) Method

Adds an item to the content index. Returns a Promise that resolves when complete.

index.delete(id) Method

Removes an item from the content index by ID. Returns a Promise.

index.getAll() Method

Returns a Promise that resolves with an array of all indexed content descriptions.

ContentDescription Interface

id Property

Unique identifier for the content. Required.

title Property

Title of the content. Required.

description Property

Description of the content. Required.

url Property

URL to the content within the app's scope. Required.

category Property

Category of content: 'homepage', 'articles', 'videos', or 'audio'. Optional.

icons Property

Array of icon definitions with src, sizes, and type. Optional.

ContentCategory Enum

"homepage" Value

Main homepage content.

"articles" Value

Article or blog post content.

"videos" Value

Video content.

"audio" Value

Audio or podcast content.

Events

contentdelete Event

Fired when indexed content is deleted. Event includes id and url properties.

Usage Example

// Get Content Index from Service Worker Registration const registration = await navigator.serviceWorker.ready; const index = registration.index; // Add content to index await index.add({ id: 'article-001', title: 'My Article', description: 'A great article about web development', url: '/articles/001', category: 'articles', icons: [{ src: '/icons/article.png', sizes: '192x192', type: 'image/png' }] }); // Get all indexed content const content = await index.getAll(); console.log(content); // Delete content await index.delete('article-001');
