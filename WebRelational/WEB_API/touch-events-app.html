API 文档

触摸事件完整接口文档

Touch Events 触摸事件

EVENT touchstart

当手指触摸屏幕时触发。

element.addEventListener('touchstart', (event) => { event.preventDefault(); const touches = event.touches; console.log('触摸点数量:', touches.length); });
EVENT touchmove

当手指在屏幕上移动时触发。

element.addEventListener('touchmove', (event) => { event.preventDefault(); const touch = event.touches[0]; console.log('位置:', touch.clientX, touch.clientY); });
EVENT touchend

当手指离开屏幕时触发。

EVENT touchcancel

当触摸被取消时触发（如来电打断）。

Touch 对象属性

identifier number 触摸点的唯一标识符
target Element 触摸开始时的目标元素
clientX / clientY number 相对于视口的坐标
pageX / pageY number 相对于页面的坐标
screenX / screenY number 相对于屏幕的坐标
radiusX / radiusY number 触摸区域的椭圆半径
rotationAngle number 触摸区域的旋转角度（度）
force number 压力值 (0.0 - 1.0)

TouchEvent 接口

PROPERTY event.touches

当前屏幕上所有触摸点的列表。

// 获取所有触摸点 const touches = event.touches; for (let i = 0; i < touches.length; i++) { console.log(`触摸点 ${i}:`, touches[i].identifier); }
PROPERTY event.targetTouches

当前元素上的触摸点列表。

PROPERTY event.changedTouches

在本次事件中发生变化的触摸点列表。

// touchend 事件中获取离开的触摸点 element.addEventListener('touchend', (event) => { const changedTouches = event.changedTouches; for (let touch of changedTouches) { console.log('离开的触摸点:', touch.identifier); } });
PROPERTY event.altKey / ctrlKey / metaKey / shiftKey

触摸时修饰键的状态。

TouchList 接口

PROPERTY touchList.length

触摸点数量。

METHOD touchList.item(index)

获取指定索引的触摸点。

METHOD touchList.identifiedTouch(id)

通过标识符获取触摸点。

Pointer Events 指针事件

EVENT pointerdown / pointerup / pointermove

指针按下、抬起、移动事件。

EVENT pointerover / pointerout / pointerenter / pointerleave

指针悬停事件。

EVENT pointercancel

指针取消事件。

EVENT gotpointercapture / lostpointercapture

指针捕获获得/丢失事件。

PointerEvent 属性

pointerId number 指针唯一标识符
pointerType string 'mouse' | 'pen' | 'touch'
isPrimary boolean 是否为主指针
pressure number 压力值 (0.0 - 1.0)
width / height number 接触区域大小
tiltX / tiltY number 笔倾斜角度
twist number 笔旋转角度
tangentialPressure number 切向压力

指针捕获方法

METHOD element.setPointerCapture(pointerId)

捕获指定指针，后续事件都发送到该元素。

METHOD element.releasePointerCapture(pointerId)

释放指针捕获。

METHOD element.hasPointerCapture(pointerId)

检查是否捕获了指定指针。

Gesture Events 手势事件

EVENT gesturestart

手势开始时触发（多指触摸）。

EVENT gesturechange

手势变化时触发。

GestureEvent 属性

scale number 缩放比例（初始为1.0）
rotation number 旋转角度（度）
EVENT gestureend

手势结束时触发。

// 手势事件示例 element.addEventListener('gesturestart', (e) => { e.preventDefault(); console.log('手势开始'); }); element.addEventListener('gesturechange', (e) => { console.log('缩放:', e.scale); console.log('旋转:', e.rotation); }); element.addEventListener('gestureend', (e) => { console.log('手势结束'); });
