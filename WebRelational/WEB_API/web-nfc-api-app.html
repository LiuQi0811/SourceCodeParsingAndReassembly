API 文档

Web NFC API 完整文档

核心对象

CLASS NDEFReader

NFC 读取器对象，用于读取和写入 NDEF 格式的 NFC 标签。

CLASS NDEFMessage

NDEF 消息对象，包含一组 NDEF 记录。

CLASS NDEFRecord

NDEF 记录对象，表示单个 NFC 数据记录。

NDEFReader 方法

方法说明
scan(options)开始扫描 NFC 标签
write(message, options)向标签写入数据
onreading读取事件处理
onreadingerror读取错误事件

NDEFRecord 类型

类型说明
"text"文本记录
"url"URL 记录
"json"JSON 记录
"opaque"二进制记录
"empty"空记录

使用示例

// 读取 NFC 标签 const reader = new NDEFReader(); await reader.scan(); reader.onreading = event => { const message = event.message; for (const record of message.records) { console.log('记录类型:', record.recordType); console.log('媒体类型:', record.mediaType); if (record.recordType === 'text') { const decoder = new TextDecoder(record.encoding); console.log('文本:', decoder.decode(record.data)); } } }; // 写入 NFC 标签 const writer = new NDEFReader(); await writer.write({ records: [{ recordType: 'text', data: 'Hello NFC!', lang: 'zh-CN' }] }); // 写入 URL await writer.write({ records: [{ recordType: 'url', data: 'https://example.com' }] });

浏览器支持

重要： Web NFC API 需要 HTTPS 环境，Chrome 89+ 支持，需要用户交互触发。
