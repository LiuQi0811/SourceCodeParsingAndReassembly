🌐 Web APIs 参考手册（离线可查）

整合 MDN、Google、W3C 等权威来源，涵盖主流与实验性 Web APIs。更新时间：2025 年。

🧱 一、核心 DOM & HTML APIs
  • HTMLElement — 所有 HTML 元素的基类
  • Document — 整个页面文档对象
  • Element — 通用元素接口
  • Node / EventTarget — DOM 节点与事件基础
  • MutationObserver — 监听 DOM 变化
🎨 二、CSS & 样式 APIs
  • CSSStyleSheet — 操作样式表
  • CSSStyleDeclaration — 单个样式属性集合
  • getComputedStyle() — 获取计算后样式
  • Constructable Stylesheets — 动态创建可复用样式表
  • CSS.supports() — 检测 CSS 特性支持
🖼️ 三、图形与媒体 APIs
  • CanvasRenderingContext2D — 2D 图形绘制
  • WebGLRenderingContext — 3D 图形
  • Web Audio API — 高级音频处理
  • getUserMedia() — 访问摄像头/麦克风
  • MediaRecorder — 录制音视频流
📡 四、设备与传感器 APIs
  • Geolocation API — 获取地理位置
  • DeviceOrientationEvent — 设备方向（陀螺仪）
  • Ambient Light Sensor — 环境光传感器
  • Screen Orientation API — 锁定屏幕方向
💾 五、存储与通信 APIs
  • localStorage / sessionStorage — 简单键值存储
  • IndexedDB — 浏览器内数据库
  • BroadcastChannel — 多标签页通信
  • SharedWorker — 多页面共享后台线程
  • Web Locks API — 跨 tab/worker 的资源锁
⚙️ 六、性能与监控 APIs
  • Performance API — 页面性能数据
  • PerformanceObserver — 异步监听性能条目
  • Navigation Timing — 页面导航耗时
  • Resource Timing — 资源加载性能
🔐 七、安全与身份 APIs
  • Web Crypto API — 加密、哈希、签名
  • Credential Management API — 账号密码管理
  • Permissions API — 查询/请求权限
  • Content Security Policy (CSP) — 安全策略
📱 八、PWA 与现代 Web 应用 APIs
  • Service Worker — 后台脚本，实现离线
  • Cache API — 缓存资源
  • Web Share API — 调用系统分享面板
  • File System Access API — 读写本地文件
  • Web App Manifest — 定义 PWA 安装行为
🧪 九、实验性 & 新兴 APIs
  • EyeDropper API — 屏幕取色器
  • WebTransport — 基于 QUIC 的低延迟通信
  • WebCodecs — 音视频编解码器直访
  • VirtualKeyboard API — 控制虚拟键盘
  • Contact Picker API — 选择设备联系人
🔍 十、兼容性查询 & 聚合平台
  • MDN Web API 总览 — 所有 Web APIs 列表
  • Can I Use — 浏览器兼容性查询
  • Web Platform Tests (WPT) — 浏览器一致性测试
  • Chromium Platform Status — Chrome 新特性进度
  • Web APIs Playground — 交互式示例
