Node Interface

node.nodeType Property

Returns an unsigned short representing the type of the node (1=Element, 3=Text, 8=Comment, 9=Document, etc.)

node.nodeName Property

Returns a string containing the name of the node.

node.parentNode Property

Returns the parent node of the node.

node.childNodes Property

Returns a NodeList of child nodes.

node.appendChild(child) Method

Adds a child node as the last child of the node.

node.removeChild(child) Method

Removes a child node from the node.

node.cloneNode(deep) Method

Returns a clone of the node.

Element Interface

element.id Property

Returns/sets the element's id attribute.

element.className Property

Returns/sets the element's class attribute.

element.classList Property

Returns a DOMTokenList of the element's classes.

element.getAttribute(name) Method

Returns the value of the named attribute.

element.setAttribute(name, value) Method

Sets the value of the named attribute.

element.querySelector(selectors) Method

Returns the first matching descendant element.

element.querySelectorAll(selectors) Method

Returns all matching descendant elements.

Document Interface

document.getElementById(id) Method

Returns an Element by its ID.

document.querySelector(selectors) Method

Returns the first matching element in the document.

document.createElement(tagName) Method

Creates a new element with the given tag name.

document.createTextNode(data) Method

Creates a new text node.

document.createDocumentFragment() Method

Creates a new document fragment.

EventTarget Interface

target.addEventListener(type, listener, options) Method

Registers an event listener.

target.removeEventListener(type, listener) Method

Removes an event listener.

target.dispatchEvent(event) Method

Dispatches an event to the target.

MutationObserver Interface

new MutationObserver(callback) Constructor

Creates a new MutationObserver.

observer.observe(target, options) Method

Starts observing the target node.

observer.disconnect() Method

Stops observing all nodes.

observer.takeRecords() Method

Returns pending mutation records.

Selection & Range

window.getSelection() Method

Returns the current Selection object.

document.createRange() Method

Creates a new Range object.
