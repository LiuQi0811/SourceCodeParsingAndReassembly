API 文档

XMLHttpRequest API 完整文档

核心方法

METHOD open(method, url, async, user, password)

初始化请求参数。

METHOD send(body)

发送请求。body 可选，用于 POST/PUT 请求。

METHOD abort()

中止当前请求。

METHOD setRequestHeader(name, value)

设置请求头。必须在 open() 之后、send() 之前调用。

属性

属性类型说明
readyStatenumber请求状态 (0-4)
statusnumberHTTP 状态码
statusTextstring状态文本
responseTextstring响应文本
responseany响应体
responseTypestring响应类型
timeoutnumber超时时间 (ms)

readyState 状态

值常量说明
0UNSENT未初始化
1OPENED已打开
2HEADERS_RECEIVED已接收响应头
3LOADING加载中
4DONE完成
