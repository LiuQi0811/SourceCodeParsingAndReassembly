Range 几何 API

Range 接口提供的选区几何信息

📝 文本选区演示

💡
操作提示
请在下方文本中选择一段文字，系统将显示选区的几何信息

CSSOM View Module 是 CSS Object Model 的一部分，它定义了一系列 API 来获取和操作文档的视觉视图信息。这些 API 包括元素的几何信息（如位置和尺寸）、滚动行为、以及鼠标事件的坐标系统。

Range 接口的 getBoundingClientRect() 和 getClientRects() 方法允许开发者获取文本选区的精确位置和尺寸信息，这对于实现富文本编辑器、批注功能或文本高亮效果非常有用。

通过这些 API，开发者可以创建更加智能和响应式的用户界面，例如根据选区位置显示上下文菜单、工具栏或弹出层。
