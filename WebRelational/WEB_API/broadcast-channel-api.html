BroadcastChannel Interface

new BroadcastChannel(name) Constructor

Creates a new BroadcastChannel object that allows communication on the named channel.

name string - The name of the channel to join
channel.name Property (readonly)

Returns the name of the channel as a string.

channel.postMessage(message) Method

Sends a message to all other contexts that are listening on the same channel. The message can be any structured cloneable type.

message any - The message to send (must be structured cloneable)
channel.close() Method

Closes the channel, indicating that the context will no longer receive messages on this channel.

channel.onmessage Event Handler

An event handler that is called when a message is received on the channel.

channel.onmessageerror Event Handler

An event handler that is called when a message arrives that cannot be deserialized.

MessageEvent Interface

event.data Property (readonly)

Returns the data sent by the message emitter.

event.origin Property (readonly)

Returns the origin of the message. For BroadcastChannel, this is typically empty.

event.lastEventId Property (readonly)

Returns a string representing the unique ID of the last event.

event.source Property (readonly)

Returns a reference to the message source. For BroadcastChannel, this is typically null.

event.ports Property (readonly)

Returns an array of MessagePort objects.

event.type Property (readonly)

Returns the type of the event ('message' or 'messageerror').

Events

message Event

Fired when a message is received on the channel. The event object is a MessageEvent.

messageerror Event

Fired when a message is received that cannot be deserialized.

Supported Data Types

The postMessage() method supports all structured cloneable types:

// Primitive types channel.postMessage("Hello"); // String channel.postMessage(42); // Number channel.postMessage(true); // Boolean channel.postMessage(null); // Null // Objects and Arrays channel.postMessage({ key: "value" }); // Object channel.postMessage([1, 2, 3]); // Array // Special types channel.postMessage(new Date()); // Date channel.postMessage(new RegExp("ab+c")); // RegExp channel.postMessage(new Blob()); // Blob channel.postMessage(new ArrayBuffer(8)); // ArrayBuffer channel.postMessage(new Map()); // Map channel.postMessage(new Set()); // Set
