API 文档

Notifications API 完整文档

核心方法

STATIC Notification.requestPermission()

请求用户授权显示通知，返回 Promise。

STATIC Notification.permission

返回当前权限状态：granted、denied 或 default。

STATIC Notification.maxActions

返回通知支持的最大操作按钮数量。

构造函数

CONSTRUCTOR new Notification(title, options)

创建并显示一个通知。

NotificationOptions

属性类型说明
bodystring通知正文
iconstring图标URL
imagestring大图URL
badgestring徽章图标URL
tagstring通知标签
renotifyboolean是否重新通知
requireInteractionboolean是否需要用户交互
silentboolean是否静音
vibratearray振动模式
timestampnumber时间戳
actionsarray操作按钮
dataany自定义数据

事件

事件说明
onclick通知被点击时触发
onclose通知被关闭时触发
onerror通知发生错误时触发
onshow通知显示时触发

使用示例

// 请求权限 const permission = await Notification.requestPermission(); if (permission === 'granted') { // 创建基础通知 const notification = new Notification('标题', { body: '通知正文', icon: '/icon.png', tag: 'my-notification' }); // 点击事件 notification.onclick = (event) => { console.log('通知被点击'); window.focus(); notification.close(); }; // 关闭事件 notification.onclose = (event) => { console.log('通知被关闭'); }; } // 带操作的通知 new Notification('新消息', { body: '您有一条新消息', icon: '/icon.png', actions: [ { action: 'reply', title: '回复' }, { action: 'ignore', title: '忽略' } ] });
