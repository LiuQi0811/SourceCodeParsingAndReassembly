⚡ 快速开始

// 获取单个 Cookie const cookie = await cookieStore.get('session_id'); console.log(cookie.value); // 获取所有 Cookie const cookies = await cookieStore.getAll(); // 设置 Cookie await cookieStore.set({ name: 'theme', value: 'dark', expires: Date.now() + 86400000 }); // 删除 Cookie await cookieStore.delete('theme'); // 监听变化 cookieStore.addEventListener('change', (event) => { console.log('Changed:', event.changed); console.log('Deleted:', event.deleted); });

⚠️ 使用要求

⚠️
安全要求
Cookie Store API 需要以下条件：
  • 必须通过 HTTPS 提供页面（localhost 除外）
  • HttpOnly Cookie 不可通过此 API 访问
  • 仅能访问同源 Cookie

方法 参数 返回值 说明
get() name | options Promise<CookieListItem?> 获取单个 Cookie
getAll() name? | options? Promise<CookieListItem[]> 获取所有 Cookie
set() name, value | options Promise<void> 设置 Cookie
delete() name | options Promise<void> 删除 Cookie

CookieListItem 接口

CookieListItem 表示单个 Cookie 的信息

📋 属性

属性 类型 说明
name string Cookie 名称
value string Cookie 值
domain string Cookie 域名
path string Cookie 路径
expires number 过期时间戳
secure boolean 是否仅 HTTPS
sameSite string SameSite 属性

🏷️ SameSite 值

值 说明
"strict"仅同站请求发送
"lax"允许顶级导航发送（默认）
"none"跨站请求也发送（需 secure）

CookieChangeEvent 接口

Cookie 变化事件对象

📋 属性

属性 类型 说明
changed CookieListItem[] 已更改的 Cookie 列表
deleted CookieListItem[] 已删除的 Cookie 列表

📝 事件用法

cookieStore.addEventListener('change', (event) => { if (event.changed.length > 0) { console.log('Changed cookies:'); event.changed.forEach(cookie => { console.log(` ${cookie.name} = ${cookie.value}`); }); } if (event.deleted.length > 0) { console.log('Deleted cookies:'); event.deleted.forEach(cookie => { console.log(` ${cookie.name}`); }); } });
