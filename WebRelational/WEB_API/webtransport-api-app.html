API 文档

WebTransport API 完整文档

核心接口

CONSTRUCTOR new WebTransport(url, options)

创建一个新的 WebTransport 连接。

PROPERTY WebTransport.ready

Promise，连接就绪时解决。

PROPERTY WebTransport.closed

Promise，连接关闭时解决。

PROPERTY WebTransport.datagrams

数据报读写接口。

方法

方法说明
createUnidirectionalStream()创建单向发送流
createBidirectionalStream()创建双向流
incomingUnidirectionalStreams传入单向流
incomingBidirectionalStreams传入双向流
close()关闭连接

数据报接口

INTERFACE WebTransportDatagramDuplexStream

提供可读和可写的数据报流。

属性说明
readable可读流
writable可写流
maxDatagramSize最大数据报大小

浏览器支持

最低版本要求：
  • Chrome 97+
  • Edge 97+
  • Opera 83+
  • Firefox: 实验性支持
  • Safari: 不支持
