PressureObserver Interface

new PressureObserver(callback, options?) Constructor

Creates a new PressureObserver instance. The callback receives PressureRecord[] when pressure changes.

observer.observe(source, options?) Method

Starts observing the specified pressure source. Options can include sampleRate.

observer.unobserve(source) Method

Stops observing the specified pressure source.

observer.disconnect() Method

Stops observing all pressure sources.

observer.takeRecords() Method

Returns pending PressureRecords and clears the queue.

PressureRecord Interface

record.source Property

Returns the pressure source (e.g., "cpu").

record.state Property

Returns the current pressure state: "nominal", "fair", "serious", or "critical".

record.time Property

Returns the timestamp when the record was created.

PressureState Enum

"nominal" State

The system is under no pressure. All features can run at full capacity.

"fair" State

The system is under moderate pressure. Consider reducing non-essential work.

"serious" State

The system is under significant pressure. Reduce work significantly.

"critical" State

The system is under critical pressure. Only essential work should continue.

PressureSource

"cpu" Source

CPU pressure. Measures the overall CPU utilization and pressure on the system.

Static Methods

PressureObserver.supportedSources Property

Returns an array of supported pressure sources on this device.
