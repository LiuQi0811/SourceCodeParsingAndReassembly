API 文档

Web Serial API 完整文档

核心方法

METHOD navigator.serial.requestPort(options)

请求用户选择一个串口设备，返回 Promise<SerialPort>。

METHOD navigator.serial.getPorts()

获取已授权的串口设备列表，返回 Promise<SerialPort[]>。

SerialPort 方法

方法说明
open(options)打开串口连接
close()关闭串口连接
getInfo()获取串口信息
getSignals()获取信号状态
setSignals(signals)设置信号状态

SerialOptions

属性类型说明
baudRatenumber波特率
dataBitsnumber数据位 (7 或 8)
stopBitsnumber停止位 (1 或 2)
paritystring校验位 (none, even, odd)
bufferSizenumber缓冲区大小
flowControlstring流控制 (none, hardware)

使用示例

// 请求串口 const port = await navigator.serial.requestPort(); // 打开串口 await port.open({ baudRate: 115200, dataBits: 8, stopBits: 1, parity: 'none' }); // 读取数据 const reader = port.readable.getReader(); while (true) { const { value, done } = await reader.read(); if (done) break; console.log('收到数据:', value); } reader.releaseLock(); // 写入数据 const writer = port.writable.getWriter(); await writer.write(new Uint8Array([0x01, 0x02, 0x03])); writer.releaseLock(); // 关闭串口 await port.close();
