API 文档

WebOTP API 完整文档

核心接口

INTERFACE OTPCredential

继承自 Credential，包含 OTP 验证码信息。

PROPERTY OTPCredential.code

从短信中提取的 OTP 验证码。

主要方法

方法说明
navigator.credentials.get(options)请求 OTP 凭证

请求选项

const options = { otp: { transport: ['sms'] }, signal: abortController.signal }; const credential = await navigator.credentials.get(options); if (credential) { const otp = credential.code; console.log('Received OTP:', otp); }

短信格式要求

短信必须包含以下格式：

您的验证码是：123456

@example.com #123456

其中 example.com 是您的域名，123456 是验证码

浏览器支持

最低版本要求：
  • Chrome 84+ (Android)
  • Edge 84+ (Android)
  • Opera 70+ (Android)
  • Firefox: 不支持
  • Safari: 不支持
  • iOS: 不支持
