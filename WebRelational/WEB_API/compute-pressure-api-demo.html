API 状态 说明

⚡ 快速开始

// 创建压力观察器 const observer = new PressureObserver((changes) => { for (const change of changes) { console.log('压力源:', change.source); console.log('压力状态:', change.state); console.log('时间戳:', change.time); } }, { sampleRate: 1.0 }); // 开始观察 CPU 压力 await observer.observe('cpu'); // 停止观察 observer.disconnect();

📊 压力状态说明

✅
nominal
正常状态，系统压力低，可以正常运行所有功能
⚠️
fair
一般状态，系统压力适中，建议适当降低非关键任务
🔶
serious
严重状态，系统压力高，应显著降低工作负载
🔴
critical
危急状态，系统压力极高，应暂停非关键操作

PressureObserver 接口

PressureObserver 用于观察系统压力变化

🏗️ 构造函数

new PressureObserver(callback, options?) // callback: (changes: PressureRecord[], observer: PressureObserver) => void // options: { sampleRate?: number }

🔧 方法

方法 参数 返回值 说明
observe() source, options? Promise<void> 开始观察指定压力源
unobserve() source void 停止观察指定压力源
disconnect() 无 void 停止所有观察
takeRecords() 无 PressureRecord[] 获取待处理的压力记录
supportedSources 静态属性 ReadonlySet<string> 返回支持的压源类型
