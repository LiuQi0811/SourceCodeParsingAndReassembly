API 文档

Web Worker API 完整文档

核心接口

CLASS Worker

创建一个专用 Worker 线程。

CLASS SharedWorker

创建一个可被多个页面共享的 Worker。

CLASS MessageChannel

创建双向通信通道。

Worker 方法

方法/属性说明
postMessage(data)向 Worker 发送消息
terminate()终止 Worker
onmessage消息接收事件
onerror错误事件
onmessageerror消息序列化错误

WorkerGlobalScope

方法/属性说明
selfWorker 全局对象
postMessage(data)向主线程发送消息
close()关闭 Worker
importScripts()导入脚本
onmessage消息接收事件
onerror错误事件

使用示例

// 创建 Worker const worker = new Worker('worker.js'); // 发送消息 worker.postMessage({ command: 'start', data: 123 }); // 接收消息 worker.onmessage = function(e) { console.log('收到结果:', e.data); }; // 错误处理 worker.onerror = function(e) { console.error('Worker 错误:', e.message); }; // 终止 Worker worker.terminate(); // Worker 内部代码 (worker.js) self.onmessage = function(e) { const result = heavyComputation(e.data); self.postMessage(result); }; // MessageChannel 示例 const channel = new MessageChannel(); channel.port1.onmessage = (e) => console.log('Port1:', e.data); channel.port2.onmessage = (e) => console.log('Port2:', e.data); channel.port1.postMessage('Hello from port1'); channel.port2.postMessage('Hello from port2');
