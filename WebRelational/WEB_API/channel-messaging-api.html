Transferable Objects

Transferable objects can be transferred (not copied) between contexts. After transfer, the original becomes unusable.

// Transferable objects demonstration // ArrayBuffer, MessagePort, ImageBitmap, etc. // After transfer, the original is neutered (unusable)
Transfer vs Copy
// Without transfer (copy): port.postMessage({ data: arrayBuffer }); // arrayBuffer is still usable here // With transfer (zero-copy): port.postMessage({ data: arrayBuffer }, [arrayBuffer]); // arrayBuffer is neutered (byteLength = 0) after transfer // Benefits of transfer: // 1. Zero-copy - more efficient // 2. No serialization overhead // 3. Instant transfer regardless of size
BroadcastChannel API

BroadcastChannel allows communication between browsing contexts (tabs, windows, iframes) with the same origin.

// BroadcastChannel allows one-to-many communication // All tabs/windows with the same channel name receive messages const bc = new BroadcastChannel('my-channel'); bc.postMessage({ hello: 'world' }); bc.onmessage = (event) => console.log(event.data);

MessageChannel Interface

new MessageChannel() Constructor

Creates a new MessageChannel with two connected ports.

channel.port1 Property

Returns the first MessagePort of the channel.

channel.port2 Property

Returns the second MessagePort of the channel.

MessagePort Interface

port.postMessage(message, transferList?) Method

Sends a message through the channel. Optional transferList for transferable objects.

port.start() Method

Starts the dispatch of messages queued for the port. Required when using addEventListener.

port.close() Method

Disconnects the port, preventing further message dispatch.

port.onmessage Event Handler

Event handler for message events.

port.onmessageerror Event Handler

Event handler for messageerror events (deserialization failures).

MessageEvent Interface

event.data Property

The message data sent by the emitter.

event.origin Property

The origin of the message emitter.

event.source Property

A reference to the message emitter (Window, MessagePort, etc.).

event.ports Property

Array of MessagePort objects sent with the message.

event.lastEventId Property

A unique ID for the event.

BroadcastChannel Interface

new BroadcastChannel(name) Constructor

Creates a new BroadcastChannel with the given name.

channel.name Property

Returns the channel name.

channel.postMessage(message) Method

Sends a message to all listeners on this channel.

channel.close() Method

Closes the channel, indicating it won't receive any more messages.

channel.onmessage / onmessageerror Event Handlers

Event handlers for message and messageerror events.

Transferable Interface

ArrayBuffer Transferable

Raw binary data buffer. Can be transferred between contexts.

MessagePort Transferable

Can be transferred to another context for direct communication.

ImageBitmap Transferable

Bitmap image data that can be transferred.

ReadableStream / WritableStream Transferable

Streams can be transferred between contexts.
