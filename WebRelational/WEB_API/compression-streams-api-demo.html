CompressionStream 接口

CompressionStream 是一个用于压缩数据的 TransformStream

🏗️ 构造函数

new CompressionStream(format: 'gzip' | 'deflate' | 'deflate-raw')
ℹ️
格式说明
gzip: GZIP 格式（带头部和校验）
deflate: DEFLATE 格式（带 zlib 头部）
deflate-raw: 原始 DEFLATE（无头部）

📋 属性

readable ReadableStream
压缩后的可读流
writable WritableStream
待压缩的可写流

DecompressionStream 接口

DecompressionStream 是一个用于解压数据的 TransformStream

🏗️ 构造函数

new DecompressionStream(format: 'gzip' | 'deflate' | 'deflate-raw')
⚠️
注意事项
解压时使用的格式必须与压缩时的格式一致，否则会抛出错误

📋 属性

解压后的可读流

GZIP 压缩格式

GZIP 是最常用的压缩格式，包含文件头和校验信息

📦 GZIP 特点

特性 值 说明
魔数 1F 8B 文件开头标识
压缩方法 DEFLATE 内部使用 DEFLATE 算法
CRC32 校验 包含 数据完整性校验
原始大小 包含 文件末尾记录原始大小
浏览器支持 广泛 HTTP 压缩标准格式
