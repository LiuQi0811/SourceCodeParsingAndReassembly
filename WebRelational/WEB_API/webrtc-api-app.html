API 文档

WebRTC API 完整文档

核心接口

INTERFACE RTCPeerConnection

WebRTC 连接的核心接口，用于建立点对点连接。

INTERFACE MediaStream

媒体流，包含音频和/或视频轨道。

INTERFACE MediaStreamTrack

单个媒体轨道（音频或视频）。

INTERFACE RTCSessionDescription

会话描述，包含 SDP 信息。

INTERFACE RTCIceCandidate

ICE 候选，描述可能的连接方法。

INTERFACE RTCDataChannel

数据通道，用于点对点数据传输。

主要方法

方法说明
navigator.mediaDevices.getUserMedia()获取用户媒体
navigator.mediaDevices.getDisplayMedia()获取屏幕共享
navigator.mediaDevices.enumerateDevices()枚举设备
new RTCPeerConnection()创建连接
peerConnection.createOffer()创建 Offer
peerConnection.createAnswer()创建 Answer
peerConnection.setLocalDescription()设置本地描述
peerConnection.setRemoteDescription()设置远程描述
peerConnection.addIceCandidate()添加 ICE 候选
peerConnection.createDataChannel()创建数据通道
peerConnection.addTrack()添加媒体轨道
