FontFace Interface

new FontFace(family, source, descriptors) Constructor

Creates a new FontFace object.

family string - The font family name
source string | ArrayBuffer | TypedArray - The font source
descriptors object - Optional font descriptors
fontFace.family Property

Gets or sets the font family name.

fontFace.style Property

Gets or sets the font style ('normal', 'italic', 'oblique').

fontFace.weight Property

Gets or sets the font weight ('normal', 'bold', or 100-900).

fontFace.stretch Property

Gets or sets the font stretch value.

fontFace.unicodeRange Property

Gets or sets the range of Unicode code points.

fontFace.variant Property

Gets or sets the font variant.

fontFace.featureSettings Property

Gets or sets OpenType feature settings.

fontFace.variationSettings Property

Gets or sets font variation settings.

fontFace.display Property

Gets or sets the font display strategy.

fontFace.status Property (readonly)

Returns the font loading status ('unloaded', 'loading', 'loaded', 'error').

fontFace.loaded Property (readonly)

Returns a Promise that resolves when the font is loaded.

fontFace.load() Method

Loads the font. Returns a Promise that resolves with the FontFace.

FontFaceSet Interface

document.fonts Property

Returns the FontFaceSet for the document.

fontFaceSet.size Property (readonly)

Returns the number of FontFace objects in the set.

fontFaceSet.status Property (readonly)

Returns the loading status ('loading', 'loaded').

fontFaceSet.ready Property (readonly)

Returns a Promise that resolves when all fonts are loaded.

fontFaceSet.onloading Event Handler

Called when font loading begins.

fontFaceSet.onloadingdone Event Handler

Called when all fonts finish loading.

fontFaceSet.onloadingerror Event Handler

Called when a font fails to load.

fontFaceSet.add(fontFace) Method

Adds a FontFace to the set.

fontFaceSet.delete(fontFace) Method

Removes a FontFace from the set. Returns true if removed.

fontFaceSet.clear() Method

Removes all FontFace objects from the set.

fontFaceSet.has(fontFace) Method

Returns true if the set contains the FontFace.

fontFaceSet.check(font, text) Method

Returns true if all fonts in the given font specification are loaded.

fontFaceSet.load(font, text) Method

Loads all fonts matching the font specification. Returns Promise<FontFace[]>.

fontFaceSet.forEach(callback) Method

Executes a callback for each FontFace in the set.

fontFaceSet.entries() Method

Returns an iterator for [FontFace, FontFace] pairs.

fontFaceSet.keys() Method

Returns an iterator for FontFace objects.

fontFaceSet.values() Method

Returns an iterator for FontFace objects.

FontFaceSetLoadEvent Interface

event.fontfaces Property (readonly)

Returns an array of FontFace objects affected by the event.

Complete Example

// Create a new FontFace const fontFace = new FontFace( 'MyCustomFont', 'url(https://example.com/font.woff2)', { style: 'normal', weight: '400', display: 'swap' } ); // Load the font await fontFace.load(); // Add to document.fonts document.fonts.add(fontFace); // Wait for all fonts to be ready await document.fonts.ready; // Check if a font is loaded const isLoaded = document.fonts.check('16px MyCustomFont'); // Use the font in CSS // element.style.fontFamily = 'MyCustomFont, sans-serif';
