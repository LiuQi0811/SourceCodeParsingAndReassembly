WebVTT API 完整文档

核心接口

INTERFACE VTTCue

表示单个字幕提示。

INTERFACE VTTRegion

定义字幕显示区域。

INTERFACE TextTrack

文本轨道接口。

INTERFACE TextTrackCueList

字幕提示列表。

VTTCue 属性

属性类型说明
startTimedouble开始时间（秒）
endTimedouble结束时间（秒）
textstring字幕文本
alignstring对齐方式
linestring/number垂直位置
positionstring/number水平位置
sizenumber宽度百分比
verticalstring文字方向
regionVTTRegion关联区域

VTTRegion 属性

属性类型说明
idstring区域标识
widthnumber宽度百分比
linesnumber显示行数
regionAnchorX/Ynumber区域锚点
viewportAnchorX/Ynumber视口锚点
scrollstring滚动方式

WebVTT 文件格式

WEBVTT NOTE 这是一个注释 00:00:00.000 --> 00:00:05.000 align:center 第一条字幕文本 00:00:05.000 --> 00:00:10.000 第二条字幕文本 可以有多行
