Clipboard 接口

Clipboard 接口提供了读写系统剪贴板的方法

🔧 方法

方法 参数 返回值 说明
writeText() text: string Promise<void> 写入纯文本到剪贴板
readText() 无 Promise<string> 从剪贴板读取纯文本
write() items: ClipboardItem[] Promise<void> 写入多种数据类型到剪贴板
read() 无 Promise<ClipboardItem[]> 从剪贴板读取多种数据类型

📍 访问方式

navigator.clipboard Clipboard
通过 navigator 对象访问

ClipboardItem 接口

ClipboardItem 表示一个剪贴板项，可以包含多种数据类型

🏗️ 构造函数

new ClipboardItem(data: Record<DOMString, Blob>)
ℹ️
支持的数据类型
text/plain, text/html, image/png, image/jpeg, image/gif 等

🔧 方法

方法 参数 返回值 说明
getType() type: string Promise<Blob> 获取指定 MIME 类型的数据
types 无（属性） Array<string> 返回支持的所有 MIME 类型
presentationStyle 无（属性） string 返回展示样式（unspecified/inline/attachment）

兼容方案

使用 document.execCommand 的传统剪贴板操作方法

⚠️ 已废弃警告

⚠️
已废弃 API
document.execCommand 已被废弃，建议使用现代 Clipboard API。以下仅作为兼容性参考。
命令 状态 说明
document.execCommand('copy') 已废弃 复制选中内容
document.execCommand('cut') 已废弃 剪切选中内容
document.execCommand('paste') 大多不支持 粘贴内容（安全限制）

🔧 兼容演示

📝 兼容代码示例

// 兼容性复制函数 function fallbackCopyText(text) { const textarea = document.createElement('textarea'); textarea.value = text; textarea.style.position = 'fixed'; textarea.style.opacity = '0'; document.body.appendChild(textarea); textarea.select(); try { document.execCommand('copy'); return true; } catch (err) { return false; } finally { document.body.removeChild(textarea); } } // 优先使用现代 API async function copyText(text) { if (navigator.clipboard?.writeText) { await navigator.clipboard.writeText(text); } else { fallbackCopyText(text); } }
