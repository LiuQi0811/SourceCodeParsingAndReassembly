API 文档

Web Audio API 完整文档

核心对象

CONTEXT new AudioContext()

创建音频上下文，所有音频操作的入口点。

音频节点

节点说明
OscillatorNode生成周期性波形信号
GainNode控制音量/增益
BiquadFilterNode双二阶滤波器
AnalyserNode频谱分析
AudioBufferSourceNode播放音频缓冲
DelayNode延迟效果
DynamicsCompressorNode动态压缩
PannerNode空间音频定位
StereoPannerNode立体声声像
ConvolverNode卷积混响
WaveShaperNode波形整形

AudioContext 属性

属性说明
currentTime当前时间（秒）
sampleRate采样率
state状态 (running/suspended/closed)
destination音频输出目标

使用示例

// 创建音频上下文 const audioContext = new AudioContext(); // 创建振荡器 const oscillator = audioContext.createOscillator(); oscillator.type = 'sine'; oscillator.frequency.value = 440; // 创建增益节点 const gainNode = audioContext.createGain(); gainNode.gain.value = 0.5; // 连接节点 oscillator.connect(gainNode); gainNode.connect(audioContext.destination); // 播放 oscillator.start(); oscillator.stop(audioContext.currentTime + 1); // 创建分析器 const analyser = audioContext.createAnalyser(); analyser.fftSize = 2048; const dataArray = new Uint8Array(analyser.frequencyBinCount); // 获取频谱数据 analyser.getByteFrequencyData(dataArray);
