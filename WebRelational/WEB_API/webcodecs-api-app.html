API 文档

WebCodecs API 完整文档

核心接口

CLASS VideoEncoder

视频编码器，将 VideoFrame 编码为 EncodedVideoChunk。

CLASS VideoDecoder

视频解码器，将 EncodedVideoChunk 解码为 VideoFrame。

CLASS AudioEncoder

音频编码器，将 AudioData 编码为 EncodedAudioChunk。

CLASS AudioDecoder

音频解码器，将 EncodedAudioChunk 解码为 AudioData。

编解码器通用方法

方法说明
configure(config)配置编解码器
encode(frame/data)编码帧/数据
decode(chunk)解码块
flush()刷新缓冲区
close()关闭编解码器
reset()重置编解码器
isConfigSupported(config)静态方法，检查配置支持

VideoFrame 属性/方法

属性/方法说明
format像素格式
codedWidth/codedHeight编码尺寸
displayWidth/displayHeight显示尺寸
timestamp时间戳
duration持续时间
allocationSize()获取所需内存大小
copyTo(buffer)复制数据到缓冲区
close()关闭帧

使用示例

// 创建视频编码器 const encoder = new VideoEncoder({ output: (chunk, metadata) => { console.log('编码块:', chunk); }, error: (e) => console.error(e) }); // 配置编码器 await encoder.configure({ codec: 'avc1.42E01E', width: 640, height: 360, bitrate: 1000000, framerate: 30 }); // 创建视频帧并编码 const canvas = document.createElement('canvas'); canvas.width = 640; canvas.height = 360; const ctx = canvas.getContext('2d'); ctx.fillStyle = 'red'; ctx.fillRect(0, 0, 640, 360); const frame = new VideoFrame(canvas, { timestamp: 0 }); encoder.encode(frame); frame.close(); // 刷新并关闭 await encoder.flush(); encoder.close(); // 创建视频解码器 const decoder = new VideoDecoder({ output: (frame) => { console.log('解码帧:', frame); frame.close(); }, error: (e) => console.error(e) }); await decoder.configure({ codec: 'avc1.42E01E', codedWidth: 640, codedHeight: 360 });
