EventSource API 完整参考

构造函数 new EventSource()

创建一个新的 EventSource 实例，用于接收服务器发送的事件。支持配置跨域凭证等选项。

// 基础用法
const source = new EventSource('/api/events');

// 带配置选项
const source = new EventSource('/api/events', {
  withCredentials: true
});

只读属性 source.url

返回创建 EventSource 时使用的 URL 字符串，包含完整的请求地址。

const source = new EventSource('https://api.example.com/sse');

console.log(source.url);
// 输出: "https://api.example.com/sse"

只读属性 source.readyState

返回连接的当前状态。0=CONNECTING, 1=OPEN, 2=CLOSED。可用于监控连接生命周期。

const STATE = {
  CONNECTING: 0, // 正在连接
  OPEN: 1,        // 已连接
  CLOSED: 2       // 已关闭
};

if (source.readyState === STATE.OPEN) {
  console.log('连接已建立');
}

只读属性 source.withCredentials

返回布尔值，指示是否在跨域请求中包含 Cookie 和认证信息。需要在构造时设置。

const source = new EventSource('https://api.example.com/sse', {
  withCredentials: true
});

console.log(source.withCredentials); // true

事件 source.onopen

当连接成功建立时触发。可用于初始化UI状态、记录日志或执行连接后的操作。

source.onopen = (event) => {
  console.log('SSE 连接已建立');
  updateConnectionStatus('connected');
};

// 或使用 addEventListener
source.addEventListener('open', handler);

事件 source.onmessage

当收到默认消息事件时触发。event.data 包含消息内容，event.origin 包含来源。

source.onmessage = (event) => {
  console.log('收到消息:', event.data);
  console.log('来源:', event.origin);
  console.log('最后事件ID:', event.lastEventId);
};

事件 source.onerror

当发生错误时触发。常见原因包括网络中断、服务器错误、CORS问题等。浏览器会自动尝试重连。

source.onerror = (event) => {
  console.error('SSE 错误:', event);
  if (source.readyState === EventSource.CLOSED) {
    console.log('连接已关闭');
  } else {
    console.log('正在重连...');
  }
};

事件 source.addEventListener()

监听自定义事件类型。服务器可以发送带有 event 字段的自定义事件，客户端通过此方法监听。

// 服务器发送格式:
// event: notification
// data: {"message": "Hello"}

source.addEventListener('notification', (e) => {
  const data = JSON.parse(e.data);
  console.log('通知:', data.message);
});

方法 source.close()

关闭 SSE 连接，停止接收事件。调用后 readyState 变为 CLOSED，浏览器不再尝试重连。

// 主动关闭连接
source.close();

console.log(source.readyState); // 2 (CLOSED)

// 清理引用
source = null;

对象 MessageEvent 属性

消息事件对象包含多个有用的属性，用于获取消息详情和来源信息。

source.onmessage = (event) => {
  event.data        // 消息数据内容
  event.origin      // 消息来源域名
  event.lastEventId // 最后事件ID
  event.source     // 发送者引用
  event.ports      // 消息端口数组
};
