ClipboardItem (Multiple Formats)

ClipboardItem allows writing multiple MIME types to the clipboard at once.

// ClipboardItem supports multiple MIME types const item = new ClipboardItem({ 'text/plain': new Blob(['text'], { type: 'text/plain' }), 'text/html': new Blob(['html'], { type: 'text/html' }) }); await navigator.clipboard.write([item]);

Clipboard Interface

navigator.clipboard.readText() Method

Returns a Promise that resolves with the text content of the clipboard. Requires clipboard-read permission.

navigator.clipboard.writeText(text) Method

Writes text to the clipboard. Returns a Promise that resolves when complete.

navigator.clipboard.read() Method

Returns a Promise that resolves with an array of ClipboardItem objects. Requires clipboard-read permission.

navigator.clipboard.write(items) Method

Writes ClipboardItem objects to the clipboard. Returns a Promise.

ClipboardItem Interface

new ClipboardItem(data) Constructor

Creates a new ClipboardItem with the given MIME type to Blob mapping.

item.types Property

Returns an array of MIME types available in the ClipboardItem.

item.getType(type) Method

Returns a Promise that resolves with a Blob for the specified MIME type.

item.presentationStyle Property

Returns the presentation style: 'unspecified', 'inline', or 'attachment'.

ClipboardEvent Interface

event.clipboardData Property

Returns a DataTransfer object containing clipboard data.

copy / cut / paste events Events

Fired on copy, cut, and paste operations. Can be prevented with event.preventDefault().
