Navigation API 简介

Navigation API 是现代浏览器提供的一套完整的导航管理接口，用于替代传统的 History API。 它提供了更强大的功能，包括导航拦截、更精细的历史记录控制、以及完善的事件系统。

// 检查浏览器支持 if (navigation in window) { // 获取当前条目 const entry = navigation.currentEntry; console.log(entry.url, entry.state); // 监听导航事件 navigation.addEventListener('navigate', (e) => { console.log('Navigating to:', e.destination.url); }); }
