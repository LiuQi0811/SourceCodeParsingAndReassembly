✅ 使用建议

  • 初学者：从 RealWorld 或 TodoMVC 开始，跑通 → 修改 → 扩展功能。
  • 进阶者：克隆 Jira Clone 或 Excalidraw，尝试添加新特性（如导出、历史记录）。
  • 面试准备：重点研究 Outline 或 HospitalRun 的目录结构与状态管理方式。
  • 工程化提升：对比不同 RealWorld 实现（Redux vs Zustand vs React Query）。
  • 探索新方向：尝试 AI 集成（如 Chatbot UI）或低代码平台（如 Appsmith）。
