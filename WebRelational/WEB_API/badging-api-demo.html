API Introduction

The Badging API gives web developers a method of setting a badge on a document or application, to act as a notification that state has changed without displaying a more distracting notification.

🎯 Core Features

  • ✓ Set badges on app icons
  • ✓ Display numeric or flag badges
  • ✓ Clear badges when not needed
  • ✓ Works in Service Workers
  • ✓ Less intrusive than notifications
  • ✓ Cross-platform support

📦 API Components

Navigator.setAppBadge() Method
Navigator.clearAppBadge() Method
WorkerNavigator.setAppBadge() Method
WorkerNavigator.clearAppBadge() Method

🏷️ Badge States

State Value Description
nothing No badge The badge is cleared and not displayed
flag Flag badge A simple marker badge without a number
number Numeric badge A badge displaying a specific number

💻 Quick Start

// Set a numeric badge navigator.setAppBadge(12); // Set a flag badge (no number) navigator.setAppBadge(); // Clear the badge navigator.clearAppBadge(); // Using async/await async function updateBadge(count) { if (count > 0) { await navigator.setAppBadge(count); } else { await navigator.clearAppBadge(); } }

▶️ Navigator.setAppBadge()

Method

The setAppBadge() method sets a badge on the icon of this app. The method returns a Promise that resolves to undefined.

Syntax

const promise = navigator.setAppBadge(); const promise = navigator.setAppBadge(contents);

📥 Parameters

Parameter Type Description
contents number (optional) The value to display in the badge. If 0, clears the badge.

📤 Return Value

Promise<undefined> Promise

Returns a Promise that resolves to undefined when the badge has been set.

⚠️ Exceptions

Exception Description
NotAllowedError The operation is not allowed
InvalidStateError Document is not active
SecurityError Security requirements not met

💻 Examples

// Set a numeric badge const unread = 30; navigator.setAppBadge(unread); // Set a flag badge (no number) navigator.setAppBadge(); // With error handling navigator.setAppBadge(5) .then(() => console.log('Badge set!')) .catch((error) => console.error('Failed to set badge:', error)); // Using async/await async function setUnreadCount(count) { try { await navigator.setAppBadge(count); console.log('Badge updated to', count); } catch (error) { console.error('Badge update failed:', error); } }

⏹️ Navigator.clearAppBadge()

Method

The clearAppBadge() method clears the badge on the icon of this app. It is functionally equivalent to calling setAppBadge(0).

Syntax

const promise = navigator.clearAppBadge();

📥 Parameters

None void

📤 Return Value

Promise<undefined> Promise

💻 Examples

// Clear the badge navigator.clearAppBadge(); // With error handling navigator.clearAppBadge() .then(() => console.log('Badge cleared!')) .catch((error) => console.error('Failed to clear badge:', error)); // Using async/await async function clearBadge() { try { await navigator.clearAppBadge(); console.log('Badge cleared successfully'); } catch (error) { console.error('Failed to clear badge:', error); } } // Clear when all items are read function markAllAsRead() { unreadCount = 0; navigator.clearAppBadge(); }

Worker Support

The Badging API is also available in Service Workers and Web Workers through WorkerNavigator, allowing you to update badges even when the app is not in the foreground.

👷 WorkerNavigator.setAppBadge()

Method

Sets a badge from within a Worker context. This is particularly useful for Service Workers that handle push notifications.

// In a Service Worker self.addEventListener('push', (event) => { const data = event.data.json(); // Update badge count if (data.badge) { self.registration.setAppBadge(data.badge); } });

👷 WorkerNavigator.clearAppBadge()

Method

Clears the badge from within a Worker context.

// In a Service Worker self.addEventListener('notificationclick', (event) => { event.notification.close(); // Clear badge when user clicks notification self.registration.clearAppBadge(); // Open the app event.waitUntil(clients.openWindow('/')); });

💻 Complete Service Worker Example

// service-worker.js // Handle push events self.addEventListener('push', (event) => { const data = event.data ? event.data.json() : {}; const unreadCount = data.unreadCount || 1; // Update badge event.waitUntil( self.registration.setAppBadge(unreadCount) .then(() => console.log('Badge updated')) .catch(err => console.error('Badge failed:', err)) ); // Show notification event.waitUntil( self.registration.showNotification(data.title || 'New Message', { body: data.body || 'You have a new message', icon: '/icon.png' }) ); }); // Handle notification click self.addEventListener('notificationclick', (event) => { event.notification.close(); // Clear badge event.waitUntil( self.registration.clearAppBadge() .then(() => clients.openWindow('/')) ); });

Requirements

The Badging API requires: 1) HTTPS or localhost, 2) App installed as PWA (for persistent badges), 3) Cross-platform support varies. Badges work best on Windows, macOS, and Android.
