PaintWorkletGlobalScope Interface

registerPaint(name, paintCtor) Method

Registers a paint worklet with the given name. The paintCtor is a class that implements the paint interface.

CSS.paintWorklet.addModule(url) Method

Loads a paint worklet module from the specified URL. Returns a Promise.

Paint Worklet Class

static get inputProperties() Static Getter

Returns an array of CSS custom property names the worklet will read.

static get inputArguments() Static Getter

Returns an array of argument types for paint() function arguments.

static get contextOptions() Static Getter

Returns context options like { alpha: true } for the rendering context.

paint(ctx, size, properties, args) Method

Called when the element needs to be painted. ctx is PaintRenderingContext2D, size is PaintSize, properties is StylePropertyMapReadOnly, args are the paint() arguments.

PaintSize Interface

size.width Property (readonly)

The width of the painting area in pixels.

size.height Property (readonly)

The height of the painting area in pixels.

StylePropertyMapReadOnly Interface

properties.get(property) Method

Returns the CSSStyleValue for the given property, or null.

properties.getAll(property) Method

Returns an array of CSSStyleValue objects for the property.

properties.has(property) Method

Returns true if the property exists in the map.

properties.size Property (readonly)

Returns the number of properties in the map.

properties.entries() Method

Returns an iterator for [property, value] pairs.

properties.keys() Method

Returns an iterator for property names.

properties.values() Method

Returns an iterator for CSSStyleValue objects.

properties.forEach(callback) Method

Executes a callback for each property in the map.

CSS paint() Function

paint(workletName) CSS Function

Uses the registered paint worklet as a CSS image. Can be used with background, border-image, etc.

paint(workletName, arg1, arg2, ...) CSS Function

Passes arguments to the paint worklet. Arguments are accessed via the args parameter in paint().

Complete Example

// In a separate worklet file (paint-worklet.js) class CirclePainter { static get inputProperties() { return ['--circle-color', '--circle-radius']; } paint(ctx, size, properties) { const color = properties.get('--circle-color').toString(); const radius = parseFloat(properties.get('--circle-radius')) || 50; ctx.fillStyle = color; ctx.beginPath(); ctx.arc(size.width / 2, size.height / 2, radius, 0, Math.PI * 2); ctx.fill(); } } registerPaint('circle-painter', CirclePainter); // In main JavaScript CSS.paintWorklet.addModule('paint-worklet.js'); // In CSS /* .circle { --circle-color: #06b6d4; --circle-radius: 50; background: paint(circle-painter); } */
