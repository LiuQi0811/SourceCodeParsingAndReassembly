Use Cases
📺
Dual-Screen Layout
Adjust UI layout when device is folded to use both screens optimally. Show content on one screen and controls on the other.
🎮
Gaming Controls
Use folded mode as a game controller with touchpad on one screen and game display on the other.
📖
Reading Mode
In folded mode, display book content on one screen and notes or bookmarks on the other.
📊
Productivity Apps
Show spreadsheet on one screen and keyboard/input on the other when device is folded.
🎬
Video Playback
In continuous mode, use full screen for video. In folded mode, show video on one screen and controls on the other.

DevicePosture Interface

navigator.devicePosture Property

Returns the DevicePosture interface for the device. Read-only.

devicePosture.type Property

Returns the current device posture type as a DevicePostureType enum value.

devicePosture.addEventListener('change', callback) Method

Registers a callback to be called when the device posture changes.

devicePosture.removeEventListener('change', callback) Method

Removes a previously registered change event listener.

DevicePostureType Enum

"continuous" Value

The device is in a continuous posture (flat, unfolded, or traditional non-foldable device).

"folded" Value

The device is in a folded posture (partially or fully folded, creating two distinct screen areas).

DevicePostureEvent Interface

new DevicePostureEvent(type, options) Constructor

Creates a new DevicePostureEvent. The event is fired when the device posture changes.

Related CSS Media Features

@media (screen-spanning: single-fold-vertical) Media Query

Matches when the device has a single vertical fold (like Samsung Galaxy Fold).

@media (screen-spanning: single-fold-horizontal) Media Query

Matches when the device has a single horizontal fold (like Microsoft Surface Duo).

env(fold-top), env(fold-bottom), env(fold-left), env(fold-right) CSS Environment Variables

CSS environment variables that define the position of the fold on the screen.
