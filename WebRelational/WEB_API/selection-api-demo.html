Selection API 简介

Selection API 是浏览器原生提供的接口，用于访问和操作用户在文档中选择的文本范围。每个文档都有一个关联的 Selection 对象，可通过 window.getSelection() 或 document.getSelection() 获取。Selection 对象表示用户选择的文本范围或光标位置。

🔗 获取 Selection 对象
// 方式一：通过 window 对象获取 const selection = window.getSelection(); // 方式二：通过 document 对象获取 const selection = document.getSelection(); // 两种方式返回同一个 Selection 对象 console.log(window.getSelection() === document.getSelection()); // true
📊 Selection 对象结构
anchorNode → anchorOffset ⟷ focusNode → focusOffset
概念 说明
anchor (锚点) 选区的起始位置，即用户开始选择的地方
focus (焦点) 选区的结束位置，即用户结束选择的地方
collapsed (折叠) 当 anchor 和 focus 在同一位置时，选区被折叠（光标状态）
range (范围) Selection 可以包含一个或多个 Range 对象
🌐 浏览器支持
🔵 Chrome 1+
🟠 Firefox 1+
🔵 Safari 1+
🔵 Edge 12+
🔵 Opera 9+

Selection 方法

Selection 对象提供了 16 个方法，用于操作选区。这些方法可以添加、移除、修改选区，以及执行各种选区相关的操作。

🎯 获取与添加 Range
getRangeAt(index) ✓ 标准

返回选区中指定索引的 Range 对象。大多数浏览器只支持一个 Range，索引通常为 0。

const selection = window.getSelection(); if (selection.rangeCount > 0) { const range = selection.getRangeAt(0); console.log(range); }
addRange(range) ✓ 标准

将一个 Range 对象添加到选区中。在大多数浏览器中，只能添加一个 Range。

const range = document.createRange(); range.selectNodeContents(element); const selection = window.getSelection(); selection.removeAllRanges(); selection.addRange(range);
🗑️ 移除 Range
removeRange(range) ✓ 标准

从选区中移除指定的 Range 对象。

const selection = window.getSelection(); const range = selection.getRangeAt(0); selection.removeRange(range);
removeAllRanges() ✓ 标准

移除选区中的所有 Range 对象，取消所有选择。

const selection = window.getSelection(); selection.removeAllRanges();
empty() ⚠ 已废弃

removeAllRanges() 的别名，已废弃，不建议使用。

// 已废弃，请使用 removeAllRanges() 代替 selection.empty();
📍 折叠选区
collapse(node, offset) ✓ 标准

将选区折叠到指定节点的指定偏移位置。

const selection = window.getSelection(); selection.collapse(node, 0); // 折叠到节点开始 selection.collapse(node, node.length); // 折叠到节点结束
collapseToStart() ✓ 标准

将选区折叠到起始位置（anchor 位置）。

const selection = window.getSelection(); selection.collapseToStart();
collapseToEnd() ✓ 标准

将选区折叠到结束位置（focus 位置）。

const selection = window.getSelection(); selection.collapseToEnd();
🔧 修改选区
extend(node, offset) ✓ 标准

将选区的焦点移动到指定位置，保持锚点不变。

const selection = window.getSelection(); selection.extend(node, offset);
setBaseAndExtent(anchorNode, anchorOffset, focusNode, focusOffset) ✓ 标准

一次性设置选区的锚点和焦点位置。

const selection = window.getSelection(); selection.setBaseAndExtent( startNode, 0, // anchor endNode, 10 // focus );
selectAllChildren(parentNode) ✓ 标准

选中指定元素的所有子节点。

const selection = window.getSelection(); const element = document.getElementById('content'); selection.selectAllChildren(element);
🔍 查询与操作
containsNode(node, partialContainment) ✓ 标准

检查选区是否包含指定的节点。partialContainment 为 true 时，节点部分在选区内也返回 true。

const selection = window.getSelection(); const node = document.getElementById('target'); const isContained = selection.containsNode(node, true);
deleteFromDocument() ✓ 标准

从文档中删除选区内容。

const selection = window.getSelection(); selection.deleteFromDocument();
toString() ✓ 标准

返回选区中的文本内容。

const selection = window.getSelection(); const text = selection.toString(); console.log('选中的文本：', text);
setPosition(node, offset) ⚠ 别名

collapse() 的别名。

// 等同于 collapse(node, offset) selection.setPosition(node, offset);

Range 对象

Range 对象表示文档中的一个范围，包含起始点和结束点。Selection 对象通常包含一个 Range 对象。Range 提供了更细粒度的范围操作能力。

🎯 创建 Range
// 方式一：通过 document.createRange() const range1 = document.createRange(); // 方式二：通过 Selection 获取 const selection = window.getSelection(); const range2 = selection.getRangeAt(0); // 方式三：通过 Range 构造函数 const range3 = new Range();
📦 Range 属性
属性 类型 说明
startContainerNode范围起始节点
startOffsetNumber范围起始偏移量
endContainerNode范围结束节点
endOffsetNumber范围结束偏移量
collapsedBoolean范围是否折叠
commonAncestorContainerNode范围的共同祖先节点
⚙️ Range 方法
方法 说明
setStart(node, offset)设置起始位置
setEnd(node, offset)设置结束位置
setStartBefore(node)设置起始位置在节点之前
setStartAfter(node)设置起始位置在节点之后
setEndBefore(node)设置结束位置在节点之前
setEndAfter(node)设置结束位置在节点之后
selectNode(node)选择整个节点
selectNodeContents(node)选择节点内容
collapse(toStart)折叠范围
cloneContents()复制范围内容为 DocumentFragment
extractContents()提取范围内容并从文档移除
deleteContents()删除范围内容
insertNode(node)在范围起始位置插入节点
surroundContents(node)用节点包裹范围内容
cloneRange()克隆 Range 对象
toString()返回范围文本
getBoundingClientRect()获取范围边界矩形
getClientRects()获取范围所有客户端矩形

实战案例

以下是 Selection API 在实际开发中的常见应用场景和代码示例。

📌 案例1：复制文本到剪贴板 ▼
function copySelectionToClipboard() { const selection = window.getSelection(); const text = selection.toString(); if (text) { navigator.clipboard.writeText(text) .then(() => { console.log('复制成功！'); }) .catch(err => { console.error('复制失败：', err); }); } } // 监听复制事件 document.addEventListener('copy', (e) => { const selection = window.getSelection(); console.log('用户复制了：', selection.toString()); });
📌 案例2：文本高亮与保存 ▼
function highlightAndSave() { const selection = window.getSelection(); if (selection.rangeCount === 0) return; const range = selection.getRangeAt(0); const highlight = document.createElement('mark'); highlight.style.background = 'yellow'; highlight.style.padding = '2px'; try { range.surroundContents(highlight); } catch (e) { // 如果选区跨越多个元素，使用替代方案 const fragment = range.extractContents(); highlight.appendChild(fragment); range.insertNode(highlight); } // 保存高亮信息 const highlightData = { text: selection.toString(), path: getXPath(range.startContainer), timestamp: Date.now() }; // 存储到 localStorage const highlights = JSON.parse(localStorage.getItem('highlights') || '[]'); highlights.push(highlightData); localStorage.setItem('highlights', JSON.stringify(highlights)); }
📌 案例3：自定义右键菜单 ▼
const contextMenu = document.createElement('div'); contextMenu.className = 'custom-context-menu'; contextMenu.innerHTML = ` <button data-action="copy">复制</button> <button data-action="search">搜索</button> <button data-action="translate">翻译</button> `; document.body.appendChild(contextMenu); document.addEventListener('contextmenu', (e) => { const selection = window.getSelection(); const text = selection.toString(); if (text) { e.preventDefault(); contextMenu.style.display = 'block'; contextMenu.style.left = e.pageX + 'px'; contextMenu.style.top = e.pageY + 'px'; } }); document.addEventListener('click', () => { contextMenu.style.display = 'none'; });
📌 案例4：选区变化监听 ▼
let lastSelection = ''; document.addEventListener('selectionchange', () => { const selection = window.getSelection(); const text = selection.toString(); if (text !== lastSelection) { lastSelection = text; if (text.length > 0) { console.log('选中文本：', text); // 触发自定义事件 document.dispatchEvent(new CustomEvent('textselected', { detail: { text, selection } })); } } }); // 监听自定义事件 document.addEventListener('textselected', (e) => { console.log('自定义事件触发：', e.detail.text); });
📌 案例5：获取选区位置 ▼
function getSelectionCoordinates() { const selection = window.getSelection(); if (selection.rangeCount === 0) return null; const range = selection.getRangeAt(0); const rect = range.getBoundingClientRect(); return { top: rect.top + window.scrollY, left: rect.left + window.scrollX, width: rect.width, height: rect.height, centerX: rect.left + rect.width / 2, centerY: rect.top + rect.height / 2 }; } // 在选区位置显示工具栏 function showToolbarAtSelection() { const coords = getSelectionCoordinates(); if (!coords) return; toolbar.style.position = 'absolute'; toolbar.style.top = (coords.top - 50) + 'px'; toolbar.style.left = coords.centerX + 'px'; toolbar.style.display = 'block'; }
📌 案例6：文本搜索与选择 ▼
function findAndSelect(element, searchText) { const walker = document.createTreeWalker( element, NodeFilter.SHOW_TEXT, null, false ); let node; while (node = walker.nextNode()) { const index = node.textContent.indexOf(searchText); if (index !== -1) { const range = document.createRange(); range.setStart(node, index); range.setEnd(node, index + searchText.length); const selection = window.getSelection(); selection.removeAllRanges(); selection.addRange(range); // 滚动到选区 range.getBoundingClientRect().top instanceof Element && range.scrollIntoView({ behavior: 'smooth' }); return true; } } return false; }
