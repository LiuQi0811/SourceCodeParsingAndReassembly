URLPattern 构造函数

CONSTRUCTOR new URLPattern(input, baseURL?)

创建一个新的 URLPattern 实例。

// 字符串模式 const pattern = new URLPattern('https://example.com/:path'); // 对象模式 const pattern2 = new URLPattern({ hostname: 'example.com', pathname: '/:category/:id' }); // 带基础URL const pattern3 = new URLPattern('/products/:id', 'https://example.com');

test() 方法

METHOD pattern.test(input, baseURL?)

测试 URL 是否匹配模式，返回布尔值。

const pattern = new URLPattern('https://example.com/products/:id'); const result = pattern.test('https://example.com/products/123'); console.log(result); // true const result2 = pattern.test('https://example.com/users/123'); console.log(result2); // false

exec() 方法

METHOD pattern.exec(input, baseURL?)

解析 URL 并返回匹配结果对象，包含提取的参数。

const pattern = new URLPattern('https://api.example.com/:version/:resource/:id'); const result = pattern.exec('https://api.example.com/v1/users/42'); console.log(result.pathname.groups); // { version: 'v1', resource: 'users', id: '42' }
