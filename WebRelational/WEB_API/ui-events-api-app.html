API 文档

UI Events API 完整接口文档

事件继承关系

BASE Event

所有事件的基础接口。

// Event 基础属性 event.type // 事件类型 event.target // 事件目标 event.currentTarget // 当前目标 event.bubbles // 是否冒泡 event.cancelable // 是否可取消 event.timeStamp // 时间戳 event.isTrusted // 是否可信
UI UIEvent : Event

用户界面事件的基础接口。

// UIEvent 属性 event.view // Window 对象 event.detail // 事件细节
MOUSE MouseEvent : UIEvent

鼠标事件接口。

// MouseEvent 属性 event.clientX, event.clientY // 相对于视口 event.pageX, event.pageY // 相对于文档 event.screenX, event.screenY // 相对于屏幕 event.button // 按下的按钮 event.buttons // 按钮状态掩码 event.altKey, event.ctrlKey // 修饰键状态 event.shiftKey, event.metaKey // 修饰键状态 event.movementX, event.movementY // 移动距离
KEYBOARD KeyboardEvent : UIEvent

键盘事件接口。

// KeyboardEvent 属性 event.key // 按键值 (如 'a', 'Enter', 'Shift') event.code // 物理键码 (如 'KeyA', 'Enter', 'ShiftLeft') event.keyCode // 已废弃，数字键码 event.location // 按键位置 event.repeat // 是否重复 event.isComposing // 是否在组合输入中
TOUCH TouchEvent : UIEvent

触摸事件接口。

// TouchEvent 属性 event.touches // 当前所有触摸点 event.targetTouches // 目标元素上的触摸点 event.changedTouches // 发生变化的触摸点 // Touch 对象属性 touch.identifier // 唯一标识 touch.clientX, touch.clientY // 坐标 touch.pageX, touch.pageY // 页面坐标 touch.screenX, touch.screenY // 屏幕坐标
POINTER PointerEvent : MouseEvent

指针事件接口（统一鼠标、触摸、笔）。

// PointerEvent 属性 event.pointerId // 指针唯一标识 event.pointerType // 类型: 'mouse', 'pen', 'touch' event.isPrimary // 是否为主指针 event.pressure // 压力 (0-1) event.width, event.height // 接触区域大小 event.tiltX, event.tiltY // 倾斜角度 event.twist // 旋转角度
FOCUS FocusEvent : UIEvent

焦点事件接口。

// FocusEvent 属性 event.relatedTarget // 相关目标元素 // 事件类型: focus, blur, focusin, focusout
INPUT InputEvent : UIEvent

输入事件接口。

// InputEvent 属性 event.inputType // 输入类型 event.data // 输入的数据 event.isComposing // 是否在组合输入中 // inputType 值: 'insertText', 'deleteContent', 'insertFromPaste'...
DRAG DragEvent : MouseEvent

拖拽事件接口。

// DragEvent 属性 event.dataTransfer // DataTransfer 对象 // 事件类型: drag, dragstart, dragend, // dragover, dragenter, dragleave, drop // DataTransfer 方法 dataTransfer.setData(format, data) dataTransfer.getData(format) dataTransfer.clearData()
WHEEL WheelEvent : MouseEvent

滚轮事件接口。

// WheelEvent 属性 event.deltaX // 水平滚动量 event.deltaY // 垂直滚动量 event.deltaZ // Z轴滚动量 event.deltaMode // 滚动单位 // deltaMode: 0=像素, 1=行, 2=页
