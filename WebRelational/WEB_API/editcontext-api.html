EditContext Interface

new EditContext(options?) Constructor

Creates a new EditContext instance. Optional options include text and selection properties.

editContext.text Property

Returns the current text content as a string.

editContext.selectionStart Property

Returns the start position of the selection.

editContext.selectionEnd Property

Returns the end position of the selection.

editContext.characterBoundsRangeStart Property

Returns the start of the range for which character bounds are requested.

editContext.attachedElements Property

Returns a NodeList of elements this EditContext is attached to.

Methods

updateText(rangeStart, rangeEnd, text) Method

Updates the text content by replacing the specified range with new text.

updateSelection(start, end) Method

Updates the selection range.

updateControlBounds(bounds) Method

Updates the bounding rectangle of the editable control.

updateSelectionBounds(bounds) Method

Updates the bounding rectangle of the selection.

updateCharacterBounds(rangeStart, boundsList) Method

Updates the bounds for characters in the specified range.

characterBounds(index) Method

Returns the bounds for the character at the specified index.

Events

textupdate Event

Fired when the text content changes. Provides information about the change and new selection.

textformatupdate Event

Fired when the text formatting should be updated (e.g., from IME).

characterboundsupdate Event

Fired when the system needs character bounds information for rendering.

compositionstart Event

Fired when an IME composition starts.

compositionend Event

Fired when an IME composition ends.

Usage Example

// Check if EditContext is supported if ('EditContext' in window) { // Create an EditContext const editContext = new EditContext({ text: 'Hello World', selectionStart: 0, selectionEnd: 0 }); // Attach to an element const editor = document.getElementById('editor'); editor.editContext = editContext; // Listen for text updates editContext.addEventListener('textupdate', (e) => { console.log('Text changed:', e.text); console.log('Range:', e.updateRangeStart, '-', e.updateRangeEnd); renderText(); }); // Listen for character bounds requests editContext.addEventListener('characterboundsupdate', (e) => { const bounds = calculateBounds(e.rangeStart, e.rangeEnd); editContext.updateCharacterBounds(e.rangeStart, bounds); }); // Update text programmatically editContext.updateText(0, 5, 'Hi'); // Update selection editContext.updateSelection(0, 2); // Get current state console.log(editContext.text); console.log(editContext.selectionStart, editContext.selectionEnd); }
