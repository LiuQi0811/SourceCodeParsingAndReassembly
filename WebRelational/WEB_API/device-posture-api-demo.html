PostureType 枚举

定义设备可能的姿态类型

📐 姿态类型

值 说明 适用场景
"continuous" 连续模式 - 设备完全展开或为普通设备 标准单屏布局
"folded" 折叠模式 - 设备处于折叠状态 双屏/分屏布局

💡 使用建议

ℹ️
Continuous 模式
设备完全展开或为普通非折叠设备，使用标准单屏布局
ℹ️
Folded 模式
设备处于折叠状态，可能需要适配双屏或分屏显示

💡 布局策略

// 根据姿态调整布局 function updateLayout(posture) { const container = document.querySelector('.app'); if (posture === 'folded') { // 折叠模式：单栏布局 container.style.gridTemplateColumns = '1fr'; } else { // 展开模式：双栏布局 container.style.gridTemplateColumns = '1fr 1fr'; } } // 监听姿态变化 navigator.devicePosture.addEventListener('change', () => { updateLayout(navigator.devicePosture.type); });
