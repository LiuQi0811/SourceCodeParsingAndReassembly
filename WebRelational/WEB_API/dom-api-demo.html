Node 接口

Node 是所有 DOM 节点类型的基类接口

📋 属性

属性 类型 说明
nodeTypenumber节点类型
nodeNamestring节点名称
nodeValuestring?节点值
textContentstring文本内容
parentNodeNode?父节点
parentElementElement?父元素
childNodesNodeList子节点列表
firstChildNode?第一个子节点
lastChildNode?最后一个子节点
previousSiblingNode?前一个兄弟节点
nextSiblingNode?后一个兄弟节点
isConnectedboolean是否连接到文档
ownerDocumentDocument所属文档

🔧 方法

方法 参数 说明
appendChild()node添加子节点
insertBefore()node, referenceNode在参考节点前插入
removeChild()node移除子节点
replaceChild()newNode, oldNode替换子节点
cloneNode()deep?克隆节点
hasChildNodes()无是否有子节点
contains()node是否包含节点
isSameNode()node是否同一节点
isEqualNode()node是否相等节点
normalize()无规范化节点
