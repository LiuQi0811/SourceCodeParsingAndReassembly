API Introduction

The Battery Status API provides information about the system's battery charge level and lets you be notified by events that are sent when the battery level or charging status change.

🎯 Core Features

  • ✓ Get battery charge level
  • ✓ Check charging status
  • ✓ Estimate charging/discharging time
  • ✓ Real-time battery events
  • ✓ Optimize app resource usage
  • ✓ Save data before battery runs out

📦 API Components

Navigator.getBattery() Method
BatteryManager Interface
charging Property
level Property
chargingchange Event

💻 Quick Start

// Get battery information navigator.getBattery().then(function(battery) { // Current battery level (0-1) console.log('Battery level:', battery.level * 100 + '%'); // Is the battery charging? console.log('Charging:', battery.charging); // Time until fully charged (seconds) console.log('Charging time:', battery.chargingTime); // Time until empty (seconds) console.log('Discharging time:', battery.dischargingTime); // Listen for changes battery.addEventListener('levelchange', () => { console.log('Level changed:', battery.level); }); });

▶️ Navigator.getBattery()

Method

The Navigator.getBattery() method returns a Promise that resolves with a BatteryManager object which provides information about the system's battery charge level.

Syntax

const promise = navigator.getBattery();

📥 Parameters

None void

📤 Return Value

Promise<BatteryManager> Promise

A Promise that resolves with a BatteryManager object.

⚠️ Exceptions

Exception Description
SecurityError The User Agent does not expose battery information to the caller

💻 Examples

// Basic usage navigator.getBattery().then(function(battery) { console.log('Battery level:', battery.level); }); // With async/await async function checkBattery() { const battery = await navigator.getBattery(); console.log(`Battery: ${battery.level * 100}%`); } // Error handling navigator.getBattery() .then(battery => { console.log('Battery API supported'); }) .catch(error => { console.error('Battery API not supported:', error); });

📦 BatteryManager Interface

The BatteryManager interface provides information about the system's battery charge level. It is returned by the Navigator.getBattery() method.

Web IDL Definition

interface BatteryManager : EventTarget { readonly attribute boolean charging; readonly attribute unrestricted double chargingTime; readonly attribute unrestricted double dischargingTime; readonly attribute double level; attribute EventHandler onchargingchange; attribute EventHandler onchargingtimechange; attribute EventHandler ondischargingtimechange; attribute EventHandler onlevelchange; };

⚡ charging Property

Property

A boolean value indicating whether the battery is currently charging.

Type boolean
Read-only Yes

📊 level Property

Property

A double value indicating the current battery level as a value between 0.0 and 1.0.

Type double (0.0 - 1.0)
Read-only Yes

⏱️ chargingTime Property

Property

A double value indicating the amount of time remaining in seconds until the battery is fully charged, or Infinity if not charging.

Type double (seconds)

⏳ dischargingTime Property

Property

A double value indicating the amount of time remaining in seconds until the battery is completely discharged, or Infinity if charging.

Type double (seconds)
ℹ️
Battery Events

The BatteryManager interface provides events that fire when battery status changes. These events allow you to react to changes in real-time.

⚡ Event Handlers

Event Handler Description
chargingchange onchargingchange Fired when charging status changes
levelchange onlevelchange Fired when battery level changes
chargingtimechange onchargingtimechange Fired when charging time changes
dischargingtimechange ondischargingtimechange Fired when discharging time changes

💻 Examples

// Listen for all battery events navigator.getBattery().then(function(battery) { // Charging status change battery.addEventListener('chargingchange', () => { if (battery.charging) { console.log('Device is now charging'); } else { console.log('Device is now on battery'); } }); // Battery level change battery.addEventListener('levelchange', () => { console.log(`Battery level: ${battery.level * 100}%`); if (battery.level < 0.2) { console.warn('Low battery!'); } }); // Charging time change battery.addEventListener('chargingtimechange', () => { console.log(`Time to full: ${battery.chargingTime}s`); }); // Discharging time change battery.addEventListener('dischargingtimechange', () => { console.log(`Time to empty: ${battery.dischargingTime}s`); }); }); // Using event handler properties navigator.getBattery().then(battery => { battery.onlevelchange = () => { updateBatteryUI(battery.level); }; });
