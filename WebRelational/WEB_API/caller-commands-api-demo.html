调用者命令 API（Invoker Commands）演示

使用 commandfor / command 属性声明式控制 popover、dialog 与自定义命令

检测中...
示例 1：声明式弹出框（Popover）
使用 command="toggle-popover" 切换弹出框显示/隐藏。
这是一个通过 command 属性控制的弹出框。
点击按钮或通过键盘激活按钮均可触发。
示例 2：声明式对话框（Dialog）
使用 command="show-modal" / command="close" 控制模态对话框。

这是一个通过 command 控制的 <dialog>。
示例 3：自定义命令（旋转图片）
使用自定义 command="--rotate-left" / "--rotate-right"，在 JS 中通过 command 事件处理。
