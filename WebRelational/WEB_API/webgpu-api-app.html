API 文档

WebGPU API 完整文档

核心接口

INTERFACE GPUAdapter

GPU 硬件适配器，代表物理 GPU 设备。

INTERFACE GPUDevice

GPU 逻辑设备，所有 GPU 资源的创建入口。

INTERFACE GPUBuffer

GPU 内存缓冲区，用于存储顶点、索引、uniform 等数据。

INTERFACE GPUTexture

GPU 纹理资源，用于图像数据和渲染目标。

INTERFACE GPUShaderModule

WGSL 着色器模块。

INTERFACE GPURenderPipeline

渲染管线，配置完整的渲染流程。

INTERFACE GPUComputePipeline

计算管线，用于 GPU 通用计算。

主要方法

方法说明
navigator.gpu.requestAdapter()请求 GPU 适配器
adapter.requestDevice()请求 GPU 设备
device.createBuffer()创建缓冲区
device.createTexture()创建纹理
device.createShaderModule()创建着色器模块
device.createRenderPipeline()创建渲染管线
device.createComputePipeline()创建计算管线
device.createCommandEncoder()创建命令编码器
queue.submit()提交命令队列
