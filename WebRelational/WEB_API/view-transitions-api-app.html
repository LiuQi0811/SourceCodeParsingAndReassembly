API 文档

View Transitions API 完整文档

概述

API document.startViewTransition(callback)

启动视图过渡动画。callback 函数中执行 DOM 更新。

ViewTransition 对象

属性/方法类型说明
finishedPromise过渡动画完成后解析
readyPromise过渡动画准备好后解析
updateCallbackDonePromise回调函数完成后解析
skipTransition()方法跳过过渡动画

CSS 伪元素

伪元素说明
::view-transition过渡容器
::view-transition-group过渡组
::view-transition-old旧视图快照
::view-transition-new新视图快照

使用示例

// 基本用法 const transition = document.startViewTransition(() => { element.classList.toggle('active'); }); // 使用 Promise transition.ready.then(() => console.log('准备完成')); transition.finished.then(() => console.log('过渡完成')); // CSS 命名过渡 .box { view-transition-name: my-box; }
