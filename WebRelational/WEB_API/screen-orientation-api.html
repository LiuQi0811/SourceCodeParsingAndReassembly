DeviceOrientationEvent 示例 
// 监听设备方向变化
window.addEventListener('deviceorientation', (event) => {
    // Alpha: 绕Z轴旋转 (0-360°)，指南针方向
    const alpha = event.alpha;
    
    // Beta: 绕X轴旋转 (-180° to 180°)，前后倾斜
    const beta = event.beta;
    
    // Gamma: 绕Y轴旋转 (-90° to 90°)，左右倾斜
    const gamma = event.gamma;
    
    console.log(`Alpha: ${alpha}, Beta: ${beta}, Gamma: ${gamma}`);
});

// iOS 13+ 需要请求权限
if (typeof DeviceOrientationEvent.requestPermission === 'function') {
    const permission = await DeviceOrientationEvent.requestPermission();
    if (permission === 'granted') {
        // 开始监听
    }
}

API 参考

Screen Orientation API

screen.orientation
返回 ScreenOrientation 对象，包含当前屏幕方向信息
返回类型: ScreenOrientation
const orientation = screen.orientation; console.log(orientation.type); // "portrait-primary" console.log(orientation.angle); // 0
screen.orientation.type
返回当前屏幕方向的类型字符串
返回值: "portrait-primary" | "portrait-secondary" | "landscape-primary" | "landscape-secondary"
screen.orientation.angle
返回当前屏幕方向的旋转角度
返回值: 0 | 90 | 180 | 270
screen.orientation.lock(orientation)
锁定屏幕到指定方向。返回 Promise，成功时 resolve，失败时 reject
参数: orientation - "portrait" | "landscape" | "portrait-primary" | "portrait-secondary" | "landscape-primary" | "landscape-secondary" | "any" | "natural"
try { await screen.orientation.lock('portrait'); console.log('锁定成功'); } catch (error) { console.error('锁定失败:', error); }
screen.orientation.unlock()
解除屏幕方向锁定，恢复自动旋转
无参数，无返回值
screen.orientation.unlock(); console.log('已解除锁定');
screen.orientation.onchange
屏幕方向变化时触发的事件处理函数
事件类型: Event
screen.orientation.addEventListener('change', () => { console.log('方向变化:', screen.orientation.type); });

方向类型说明

portrait-primary
主纵向模式，设备处于自然纵向方向（通常角度为0°）
portrait-secondary
次纵向模式，设备倒置（通常角度为180°）
landscape-primary
主横向模式，设备向左旋转90°
landscape-secondary
次横向模式，设备向右旋转90°（或向左270°）
portrait
任意纵向模式，锁定到 portrait-primary 或 portrait-secondary
landscape
任意横向模式，锁定到 landscape-primary 或 landscape-secondary
any
允许任意方向，解除锁定效果相同
natural
自然方向，通常是设备的默认方向（手机为 portrait-primary，平板可能为 landscape-primary）

DeviceOrientationEvent

deviceorientation 事件
设备物理方向变化时触发，提供三轴旋转数据
事件属性: alpha, beta, gamma, absolute, webkitCompassHeading
window.addEventListener('deviceorientation', (e) => { // Alpha: 绕Z轴旋转 (0-360°)，指南针方向 // Beta: 绕X轴旋转 (-180° to 180°)，前后倾斜 // Gamma: 绕Y轴旋转 (-90° to 90°)，左右倾斜 console.log(e.alpha, e.beta, e.gamma); });
event.alpha
绕Z轴的旋转角度（0-360°），表示设备绕垂直轴的旋转，即指南针方向
范围: 0 - 360
event.beta
绕X轴的旋转角度（-180° to 180°），表示设备前后倾斜
范围: -180 - 180
event.gamma
绕Y轴的旋转角度（-90° to 90°），表示设备左右倾斜
范围: -90 - 90
event.absolute
指示方向数据是否为绝对值（相对于地球坐标系）
类型: boolean
DeviceOrientationEvent.requestPermission()
iOS 13+ 需要请求权限才能访问设备方向数据
返回: Promise<"granted" | "denied">
// iOS 权限请求 if (typeof DeviceOrientationEvent.requestPermission === 'function') { const permission = await DeviceOrientationEvent.requestPermission(); if (permission === 'granted') { window.addEventListener('deviceorientation', handler); } }

DeviceMotionEvent

devicemotion 事件
设备运动变化时触发，提供加速度和旋转速率数据
事件属性: acceleration, accelerationIncludingGravity, rotationRate, interval
window.addEventListener('devicemotion', (e) => { // 不含重力的加速度 console.log(e.acceleration); // 含重力的加速度 console.log(e.accelerationIncludingGravity); // 旋转速率 console.log(e.rotationRate); });
event.acceleration
设备在三轴上的加速度（不含重力）
属性: x, y, z (单位: m/s²)
event.accelerationIncludingGravity
设备在三轴上的加速度（包含重力）
属性: x, y, z (单位: m/s²)
event.rotationRate
设备在三轴上的旋转速率
属性: alpha, beta, gamma (单位: °/s)
event.interval
从设备获取数据的间隔时间（毫秒）
类型: double

相关事件

orientationchange 事件
窗口方向变化时触发（旧API，建议使用 screen.orientation.onchange）
window.addEventListener('orientationchange', () => { console.log(window.orientation); // 0, 90, -90, 180 });
resize 事件
窗口大小变化时触发，方向变化通常伴随 resize
window.addEventListener('resize', () => { console.log(window.innerWidth, window.innerHeight); });
