可见性管理

页面可见性检测
当页面变为不可见时，唤醒锁会自动释放。当页面重新可见时，可以自动重新请求唤醒锁。
自动重新获取
启用此功能后，当页面从隐藏状态恢复时，系统会自动重新请求唤醒锁，保持屏幕常亮。
安全释放机制
当用户切换标签页、最小化窗口或锁定屏幕时，唤醒锁会安全释放，避免资源浪费。

            } catch (err) {
                console.error('重新获取失败:', err);

        // 页面变为不可见，唤醒锁会自动释放
        console.log('页面不可见，唤醒锁将释放');
    }
});

使用限制

需要安全上下文
Wake Lock API 只能在安全上下文（HTTPS 或 localhost）中使用。
用户激活要求
某些浏览器要求用户交互后才能请求唤醒锁，不能在页面加载时自动请求。
跨平台行为差异
不同操作系统和浏览器对唤醒锁的处理可能有细微差异，建议进行充分测试。

API 参考

核心接口

navigator.wakeLock
WakeLock API 的入口点，用于访问唤醒锁功能
返回类型: WakeLock | undefined
if ('wakeLock' in navigator) { console.log('Wake Lock API 可用'); }
navigator.wakeLock.request(type)
请求指定类型的唤醒锁。目前只支持 'screen' 类型。
参数: type - "screen" | 返回: Promise<WakeLockSentinel>
try { const wakeLock = await navigator.wakeLock.request('screen'); console.log('唤醒锁已获取'); } catch (err) { console.error('获取失败:', err.name, err.message); }

WakeLockSentinel 对象

WakeLockSentinel
表示一个唤醒锁的句柄，用于管理和监控唤醒锁状态
接口属性和方法
sentinel.released
只读属性，指示唤醒锁是否已被释放
类型: boolean | 初始值: false
if (wakeLock.released) { console.log('唤醒锁已释放'); } else { console.log('唤醒锁仍处于活动状态'); }
sentinel.release()
手动释放唤醒锁。返回一个 Promise，在锁释放后 resolve。
返回: Promise<void>
await wakeLock.release(); console.log('唤醒锁已手动释放');
sentinel.onrelease
事件处理器，当唤醒锁被释放时触发（无论是手动还是自动）
事件类型: Event
wakeLock.addEventListener('release', () => { console.log('唤醒锁已释放'); }); // 或使用属性方式 wakeLock.onrelease = () => { console.log('唤醒锁已释放'); };
sentinel.type
只读属性，返回唤醒锁的类型
类型: "screen"
console.log(wakeLock.type); // "screen"

错误处理

NotSupportedError
浏览器不支持请求的唤醒锁类型
场景: 请求非 'screen' 类型时
SecurityError
在不安全的上下文中使用 API
场景: 非 HTTPS 或 localhost 环境
NotAllowedError
唤醒锁请求被拒绝
场景: 用户未交互、页面不可见、系统限制等
AbortError
请求被中止
场景: 页面在请求过程中变为不可见

相关事件

visibilitychange
页面可见性变化时触发，唤醒锁会在此事件后自动释放
document.addEventListener('visibilitychange', () => { if (document.visibilityState === 'visible') { // 页面可见，可以重新请求唤醒锁 } else { // 页面不可见，唤醒锁将自动释放 } });

完整示例

完整的唤醒锁管理类
class WakeLockManager { constructor() { this.wakeLock = null; this.setupVisibilityHandler(); } async request() { if (!('wakeLock' in navigator)) { throw new Error('Wake Lock API 不支持'); } try { this.wakeLock = await navigator.wakeLock.request('screen'); this.wakeLock.addEventListener('release', () => { console.log('唤醒锁已释放'); this.wakeLock = null; }); console.log('唤醒锁已获取'); return this.wakeLock; } catch (err) { console.error('获取唤醒锁失败:', err); throw err; } } async release() { if (this.wakeLock && !this.wakeLock.released) { await this.wakeLock.release(); this.wakeLock = null; } } setupVisibilityHandler() { document.addEventListener('visibilitychange', async () => { if (document.visibilityState === 'visible' && !this.wakeLock) { await this.request(); } }); } get isActive() { return this.wakeLock && !this.wakeLock.released; } } // 使用 const wakeLockManager = new WakeLockManager(); await wakeLockManager.request();
