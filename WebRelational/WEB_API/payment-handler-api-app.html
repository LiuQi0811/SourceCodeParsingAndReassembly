Service Worker 代码示例
// service-worker.js // 支付请求事件 self.addEventListener('paymentrequest', (event) => { event.respondWith( new Promise((resolve, reject) => { // 打开支付处理窗口 event.openWindow('/payment-handler') .then((client) => { // 处理支付逻辑 resolve({ methodName: event.methodData[0].supportedMethods, details: { transactionId: 'txn_' + Date.now(), status: 'success' } }); }) .catch(reject); }) ); }); // 支付能力检查事件 self.addEventListener('canmakepayment', (event) => { event.respondWith(Promise.resolve(true)); });
PaymentManager 注册示例
// 主线程代码 const registration = await navigator.serviceWorker.register('/sw.js'); // 获取 PaymentManager const paymentManager = registration.paymentManager; // 设置用户提示 paymentManager.userHint = '使用我的钱包支付'; // 添加支付工具 await paymentManager.instruments.set('my-wallet', { name: '我的钱包', icons: [{ src: '/icon.png', sizes: '96x96', type: 'image/png' }] }); // 检查工具是否存在 const hasWallet = await paymentManager.instruments.has('my-wallet'); // 获取所有工具键 const keys = await paymentManager.instruments.keys(); // 删除工具 await paymentManager.instruments.delete('my-wallet'); // 清空所有工具 await paymentManager.instruments.clear();

API 文档

Payment Handler API 完整文档

核心接口

INTERFACE PaymentManager

支付管理器，通过 ServiceWorkerRegistration.paymentManager 访问。

INTERFACE PaymentInstruments

支付工具集合，提供 Map-like 接口管理支付工具。

INTERFACE PaymentRequestEvent (Service Worker)

支付请求事件，在 Service Worker 中触发。

PaymentManager 属性

属性类型说明
instrumentsPaymentInstruments支付工具集合
userHintstring用户提示文本

PaymentInstruments 方法

方法返回值说明
set(key, instrument)Promise<void>添加/更新支付工具
get(key)Promise<PaymentInstrument>获取支付工具
has(key)Promise<boolean>检查是否存在
delete(key)Promise<boolean>删除支付工具
keys()Promise<string[]>获取所有键
clear()Promise<void>清空所有工具

PaymentRequestEvent 属性/方法

属性/方法说明
methodData支付方式数据数组
total总金额对象
paymentRequestOrigin请求来源
topOrigin顶层页面来源
openWindow(url)打开支付处理窗口
respondWith(promise)响应支付请求

浏览器支持

环境要求：
  • HTTPS 环境 (localhost 除外)
  • Service Worker 支持
  • Chrome 70+ 或 Edge 79+
  • Firefox/Safari 不支持
