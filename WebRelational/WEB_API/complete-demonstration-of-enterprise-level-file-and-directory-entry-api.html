🔄 新旧文件系统API对比

Legacy vs Modern
功能 传统API (File & Directory Entries) 现代API (File System Access)
获取文件系统 requestFileSystem() / webkitRequestFileSystem navigator.storage.getDirectory() / showOpenFilePicker
文件条目 FileEntry FileSystemFileHandle
目录条目 DirectoryEntry FileSystemDirectoryHandle
文件写入 FileWriter (createWriter) FileSystemWritableFileStream (createWritable)
存储类型 沙箱文件系统 (临时/持久) 源私有文件系统 + 用户可见文件
权限模型 基于配额，一次性请求 基于用户手势，细粒度权限
浏览器支持 Chrome, Edge, 部分移动浏览器 现代浏览器 (Chrome 86+, Edge 86+, 部分支持)
⚡ 传统API主要工作在沙箱环境，现代API可以访问用户真实文件系统
