TextEncoder Interface

new TextEncoder() Constructor

Creates a new TextEncoder instance. Always uses UTF-8 encoding.

textEncoder.encoding Property

Always returns "utf-8". Read-only.

textEncoder.encode(string) Method

Encodes a string to UTF-8. Returns a Uint8Array containing the encoded bytes.

textEncoder.encodeInto(string, destination) Method

Encodes a string into an existing Uint8Array buffer. Returns { read, written }.

TextDecoder Interface

new TextDecoder(encoding?, options?) Constructor

Creates a new TextDecoder. Supports utf-8, utf-16le, utf-16be, iso-8859-1, etc.

textDecoder.encoding Property

Returns the encoding name (lowercase). Read-only.

textDecoder.fatal Property

Boolean indicating if decoder throws on invalid sequences. Read-only.

textDecoder.ignoreBOM Property

Boolean indicating if BOM should be skipped. Read-only.

textDecoder.decode(buffer?, options?) Method

Decodes a buffer to string. Accepts ArrayBuffer, TypedArray, or DataView.

TextEncoderStream Interface

new TextEncoderStream() Constructor

Creates a stream-based UTF-8 encoder.

textEncoderStream.encoding Property

Always returns "utf-8".

textEncoderStream.readable Property

Returns the ReadableStream for encoded output.

textEncoderStream.writable Property

Returns the WritableStream for string input.

TextDecoderStream Interface

new TextDecoderStream(encoding?, options?) Constructor

Creates a stream-based decoder.

textDecoderStream.encoding Property

Returns the encoding name.

textDecoderStream.fatal Property

Returns the fatal mode setting.

textDecoderStream.ignoreBOM Property

Returns the ignoreBOM setting.

textDecoderStream.readable Property

Returns the ReadableStream for decoded output.

textDecoderStream.writable Property

Returns the WritableStream for byte input.

Usage Example

// TextEncoder - String to Bytes const encoder = new TextEncoder(); const bytes = encoder.encode('Hello, 世界!'); console.log(bytes); // Uint8Array [72, 101, 108, 108, 111, 44, 32, ...] console.log(encoder.encoding); // "utf-8" // encodeInto - Pre-allocated buffer const buffer = new Uint8Array(32); const result = encoder.encodeInto('Hello', buffer); console.log(result); // { read: 5, written: 5 } // TextDecoder - Bytes to String const decoder = new TextDecoder('utf-8'); const text = decoder.decode(bytes); console.log(text); // "Hello, 世界!" // With options const decoderFatal = new TextDecoder('utf-8', { fatal: true }); const decoderIgnoreBOM = new TextDecoder('utf-8', { ignoreBOM: true }); // TextEncoderStream const encoderStream = new TextEncoderStream(); const writer = encoderStream.writable.getWriter(); const reader = encoderStream.readable.getReader(); writer.write('Hello'); const { value } = await reader.read(); // Uint8Array // TextDecoderStream const decoderStream = new TextDecoderStream(); const decoderWriter = decoderStream.writable.getWriter(); const decoderReader = decoderStream.readable.getReader(); decoderWriter.write(new Uint8Array([72, 101, 108, 108, 111])); const { value: decoded } = await decoderReader.read(); // "Hello"
