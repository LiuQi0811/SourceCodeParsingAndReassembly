Navigation API 简介

Navigation API 是现代浏览器提供的一套完整的导航管理接口，用于替代传统的 History API。 它提供了更强大的功能，包括导航拦截、更精细的历史记录控制、以及完善的事件系统。

// 检查浏览器支持 if (navigation in window) { // Navigation API 可用 console.log(navigation.currentEntry.url); }

核心优势

  • ✅ 统一的导航入口，所有导航操作都通过 navigation 对象
  • ✅ 完整的历史条目访问，可以获取所有历史记录的详细信息
  • ✅ 强大的导航拦截能力，可以在导航发生前进行拦截和处理
  • ✅ 丰富的事件系统，实时监听导航状态变化
  • ✅ 每个历史条目独立的 state 对象

自定义导航

导航拦截器

未激活

导航拦截器可以在导航发生前进行拦截，执行自定义逻辑或取消导航。 这对于实现路由守卫、权限检查、表单保护等场景非常有用。

// 注册导航拦截器 navigation.addEventListener('navigate', (event) => { // 检查是否可以导航 if (!canNavigate) { event.preventDefault(); // 取消导航 } // 获取导航信息 console.log(event.destination.url); });
