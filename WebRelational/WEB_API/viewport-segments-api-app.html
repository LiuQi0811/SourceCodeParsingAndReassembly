API 文档

Viewport Segments API 完整文档

概述

API window.visualViewport.segments

返回一个 DOMRect 数组，表示设备屏幕上各个视口段的边界。

属性

属性类型说明
segmentsDOMRect[]视口段边界数组
widthnumber视口宽度
heightnumber视口高度
scalenumber缩放比例
offsetLeftnumber左偏移
offsetTopnumber上偏移

事件

事件说明
segmentchange视口段变化时触发
resize视口大小变化时触发
scroll视口滚动时触发

DOMRect 结构

属性说明
x, y左上角坐标
width宽度
height高度
top, right, bottom, left边界值

使用示例

// 检测视口段 if (window.visualViewport && 'segments' in visualViewport) { const segments = visualViewport.segments; console.log('视口段数量:', segments.length); segments.forEach((segment, index) => { console.log(`段 ${index + 1}:`, { x: segment.x, y: segment.y, width: segment.width, height: segment.height }); }); } // 监听变化 visualViewport.addEventListener('segmentchange', (e) => { console.log('视口段已变化'); updateLayout(visualViewport.segments); });

浏览器支持

实验性 API - Viewport Segments API 目前处于实验阶段，仅部分浏览器支持。主要用于折叠屏设备（如 Samsung Galaxy Fold、Microsoft Surface Duo）。
