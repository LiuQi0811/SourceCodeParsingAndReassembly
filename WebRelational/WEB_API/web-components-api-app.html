API 文档

Web Components API 完整文档

CustomElementRegistry

METHOD customElements.define(name, constructor, options)

注册一个新的自定义元素。

METHOD customElements.get(name)

获取已注册的自定义元素构造函数。

METHOD customElements.whenDefined(name)

返回一个 Promise，在元素定义时解析。

生命周期回调

回调说明
connectedCallback()元素插入 DOM 时调用
disconnectedCallback()元素从 DOM 移除时调用
attributeChangedCallback()属性变化时调用
adoptedCallback()元素移动到新文档时调用

Shadow DOM

方法/属性说明
attachShadow(options)创建 Shadow DOM
shadowRoot访问 Shadow Root
mode: open/closedShadow DOM 模式

`; } // 监听的属性 static get observedAttributes() { return ['data-value']; } // 生命周期回调 connectedCallback() { console.log('元素已连接'); } disconnectedCallback() { console.log('元素已断开'); } attributeChangedCallback(name, oldVal, newVal) { console.log(`属性 ${name} 从 ${oldVal} 变为 ${newVal}`); } } // 注册组件 customElements.define('my-component', MyComponent); // 使用组件 // // 标题 // 内容 //
