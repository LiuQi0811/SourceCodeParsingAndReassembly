DocumentPictureInPicture Interface

window.documentPictureInPicture Property

Returns the DocumentPictureInPicture interface. This is the main entry point for the API.

documentPictureInPicture.window Property

Returns the current Picture-in-Picture window (Window object) or null if none is open.

documentPictureInPicture.requestWindow(options) Method

Requests to open a Picture-in-Picture window. Returns a Promise that resolves to the Window object. Requires user gesture.

Window Options

width Property

The preferred width of the PiP window in pixels.

height Property

The preferred height of the PiP window in pixels.

PiP Window Properties

pipWindow.document Property

The Document object of the PiP window. Can be used to manipulate DOM.

pipWindow.close() Method

Closes the Picture-in-Picture window.

pipWindow.innerWidth / innerHeight Property

The inner dimensions of the PiP window.

Events

'pagehide' Event

Fired on the PiP window when it is about to be closed.

Usage Example

// Check if API is supported if ('documentPictureInPicture' in window) { // Open PiP window const pipWindow = await window.documentPictureInPicture.requestWindow({ width: 400, height: 300 }); // Copy styles from main document const style = document.createElement('style'); style.textContent = ` body { font-family: sans-serif; padding: 20px; } `; pipWindow.document.head.appendChild(style); // Add content const content = document.createElement('div'); content.textContent = 'Hello from PiP!'; pipWindow.document.body.appendChild(content); // Listen for close pipWindow.addEventListener('pagehide', () => { console.log('PiP window closed'); }); // Close programmatically // pipWindow.close(); } // Check if PiP window is open const currentPip = window.documentPictureInPicture.window; if (currentPip) { console.log('PiP window is open'); }
