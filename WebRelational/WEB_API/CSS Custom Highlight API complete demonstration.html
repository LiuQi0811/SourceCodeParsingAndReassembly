CSS Custom Highlight API 是一个强大的 Web API，它允许开发者创建自定义的文本高亮效果。

与传统的 ::selection 不同，它可以高亮任意范围的文本，非常适合：

  • 代码语法高亮
  • 搜索结果标记
  • 拼写错误提示
  • 重要内容突出显示

示例代码：

function hello() {
    // 这是一个注释
    const message = "Hello World";
    console.log(message);
    return message;
}

这个 API 提供了更好的性能和灵活性，是构建富文本编辑器的理想选择。
