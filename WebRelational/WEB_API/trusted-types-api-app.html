API 文档

Trusted Types API 完整接口文档

核心 API

POLICY trustedTypes.createPolicy(policyName, policyOptions)

创建一个新的 Trusted Types 策略。

const policy = trustedTypes.createPolicy('my-policy', { createHTML: (input) => input, createScript: (input) => input, createScriptURL: (input) => input });
POLICY trustedTypes.getPolicyNames()

返回已创建策略的名称数组。

const names = trustedTypes.getPolicyNames(); console.log(names); // ['my-policy', 'default', ...]
CHECK trustedTypes.getPropertyType(tagName, property)

返回指定元素属性所需的 Trusted Types 类型。

const type = trustedTypes.getPropertyType('div', 'innerHTML'); console.log(type); // 'TrustedHTML' or null
CHECK trustedTypes.getAttributeType(tagName, attribute)

返回指定元素属性所需的 Trusted Types 类型。

const type = trustedTypes.getAttributeType('script', 'src'); console.log(type); // 'TrustedScriptURL' or null

策略方法

TYPE policy.createHTML(input)

创建 TrustedHTML 对象。

const trustedHtml = policy.createHTML('
Safe HTML
'); element.innerHTML = trustedHtml;
TYPE policy.createScript(input)

创建 TrustedScript 对象。

const trustedScript = policy.createScript('console.log("Safe")'); eval(trustedScript);
TYPE policy.createScriptURL(input)

创建 TrustedScriptURL 对象。

const trustedUrl = policy.createScriptURL('https://example.com/script.js'); scriptElement.src = trustedUrl;
