API 文档

WebXR Device API 完整文档

核心接口

INTERFACE XRSystem

WebXR 的主入口，通过 navigator.xr 访问。

INTERFACE XRSession

代表一个 AR 或 VR 会话。

INTERFACE XRFrame

单帧渲染数据，包含姿态和视图信息。

INTERFACE XRView

单个视图（眼睛）的渲染信息。

INTERFACE XRViewerPose

用户的头部姿态信息。

INTERFACE XRReferenceSpace

参考空间，定义坐标系统。

INTERFACE XRRigidTransform

刚体变换，包含位置和旋转。

INTERFACE XRInputSource

输入源，代表控制器或手势。

主要方法

方法说明
navigator.xr.isSessionSupported(mode)检查会话支持
navigator.xr.requestSession(mode, options)请求会话
XRSession.requestReferenceSpace(type)请求参考空间
XRSession.requestAnimationFrame(callback)请求动画帧
XRSession.end()结束会话
XRFrame.getViewerPose(space)获取观察者姿态
XRFrame.getPose(space, baseSpace)获取空间姿态

会话模式

模式说明
immersive-vr沉浸式 VR
immersive-ar沉浸式 AR
inline内联模式（页面内）
