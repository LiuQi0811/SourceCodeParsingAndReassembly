API 文档

Translator & Language Detector API 完整接口文档

Translator API

ASYNC ai.translator.create(options)

创建一个翻译器实例。

const translator = await ai.translator.create({ sourceLanguage: 'en', targetLanguage: 'zh' });
ASYNC translator.translate(text)

翻译文本，返回翻译结果。

const result = await translator.translate('Hello, World!'); console.log(result); // "你好，世界！"

Language Detector API

ASYNC ai.languageDetector.create()

创建一个语言检测器实例。

const detector = await ai.languageDetector.create(); const results = await detector.detect('Hello, World!');
