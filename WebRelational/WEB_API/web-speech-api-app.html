语音识别 API

CLASS SpeechRecognition

语音识别对象，用于将语音转换为文本。

属性/方法说明
start()开始识别
stop()停止识别
abort()中止识别
continuous是否连续识别
interimResults是否返回中间结果
lang识别语言
maxAlternatives最大替代结果数
onresult识别结果事件
onerror错误事件
onstart开始事件
onend结束事件

语音合成 API

CLASS SpeechSynthesis

语音合成对象，用于将文本转换为语音。

属性/方法说明
speak(utterance)朗读
cancel()取消所有朗读
pause()暂停朗读
resume()继续朗读
getVoices()获取语音列表
paused是否暂停
speaking是否正在朗读

SpeechSynthesisUtterance

属性说明
text要朗读的文本
lang语言
voice语音对象
rate语速 (0.1-10)
pitch音调 (0-2)
volume音量 (0-1)

使用示例

// 语音识别 const recognition = new webkitSpeechRecognition(); recognition.continuous = true; recognition.interimResults = true; recognition.lang = 'zh-CN'; recognition.onresult = (event) => { const result = event.results[event.results.length - 1]; console.log(result[0].transcript); }; recognition.start(); // 语音合成 const utterance = new SpeechSynthesisUtterance('你好世界'); utterance.lang = 'zh-CN'; utterance.rate = 1; utterance.pitch = 1; utterance.volume = 1; // 选择语音 const voices = speechSynthesis.getVoices(); utterance.voice = voices[0]; speechSynthesis.speak(utterance);
