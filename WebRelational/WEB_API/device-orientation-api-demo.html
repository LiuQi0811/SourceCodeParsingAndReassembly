⚠️ 使用要求

⚠️
安全要求
设备方向事件 API 需要以下条件：
  • 必须通过 HTTPS 提供页面（localhost 除外）
  • iOS 13+ 需要用户手势触发权限请求
  • 移动设备支持更好，桌面设备可能返回 null

DeviceOrientationEvent 接口

表示设备的物理方向变化

📋 属性

属性 类型 范围 说明
alpha number? 0° - 360° Z轴旋转角度（罗盘方向）
beta number? -180° - 180° X轴旋转角度（前后倾斜）
gamma number? -90° - 90° Y轴旋转角度（左右倾斜）
absolute boolean -- 是否为绝对方向值
webkitCompassHeading number? 0° - 360° 罗盘方向（WebKit）
webkitCompassAccuracy number? ≥ 0 罗盘精度（WebKit）

DeviceMotionEvent 接口

表示设备的加速度和旋转速率

📋 属性

属性 类型 说明
acceleration DeviceAcceleration? 加速度 (不含重力)
accelerationIncludingGravity DeviceAcceleration? 加速度 (含重力)
rotationRate DeviceRotationRate? 旋转速率
interval number? 数据间隔时间 (ms)

属性 说明

yY轴加速度 (m/s²)
