API 状态 说明

⚡ 快速开始

// 创建 EditContext const editContext = new EditContext({ text: 'Hello World', selectionStart: 0, selectionEnd: 0 }); // 关联到元素 element.attachEditContext(editContext); // 监听文本更新 editContext.addEventListener('textupdate', (e) => { console.log(`文本更新: ${e.text}`); console.log(`位置: ${e.updateRangeStart} - ${e.updateRangeEnd}`); }); // 获取/设置文本 console.log(editContext.text); // 当前文本 editContext.updateText(0, 5, 'Hi'); // 替换文本

EditContext 接口

EditContext 是文本编辑的核心接口

🔧 构造函数

new EditContext(options?)
选项 类型 默认值 说明
textstring""初始文本内容
selectionStartnumber0选择起始位置
selectionEndnumber0选择结束位置

📋 属性

属性 类型 说明
textstring当前文本内容
selectionStartnumber选择起始位置
selectionEndnumber选择结束位置
compositionRangeStartnumber?组合范围起始
compositionRangeEndnumber?组合范围结束
isInCompositionboolean是否在 IME 组合中
controlBoundsDOMRect?编辑控件边界

🔧 方法

方法 参数 说明
updateText()rangeStart, rangeEnd, text更新指定范围的文本
updateSelection()start, end更新选择范围
updateControlBounds()bounds更新控件边界
updateCompositionRange()start, end更新组合范围
characterBounds()rangeStart, rangeEnd获取字符边界

📖 IME 组合过程

// IME 组合流程: 1. compositionstart - 用户开始输入法组合 2. textupdate - 组合过程中的文本更新 3. characterboundsupdate - 请求字符边界 4. compositionend - 组合完成 // 组合期间: - editContext.isInComposition === true - compositionRangeStart/end 标记组合范围 - 文本暂时显示为组合状态（通常有下划线）
