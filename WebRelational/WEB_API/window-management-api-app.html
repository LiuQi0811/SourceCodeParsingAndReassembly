API 文档

Window Management API 完整文档

核心接口

INTERFACE ScreenDetails

包含所有可用屏幕的信息。

INTERFACE ScreenDetailed

单个屏幕的详细信息。

METHOD window.getScreenDetails()

获取所有屏幕的详细信息，返回 Promise<ScreenDetails>。

PROPERTY screen.isExtended

布尔值，指示是否有多个屏幕。

ScreenDetailed 属性

属性类型说明
widthnumber屏幕宽度
heightnumber屏幕高度
leftnumber屏幕左边界坐标
topnumber屏幕上边界坐标
isPrimaryboolean是否为主屏幕
isInternalboolean是否为内置屏幕
devicePixelRationumber设备像素比

窗口控制方法

方法说明
window.moveTo(x, y)移动窗口到指定位置
window.resizeTo(w, h)调整窗口大小
window.open(url, name, features)打开新窗口
window.close()关闭窗口

浏览器支持

最低版本要求：

  • Edge 100+
  • Opera 86+
  • Firefox: 不支持
  • Safari: 不支持
