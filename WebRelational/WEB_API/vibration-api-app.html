API 文档

Vibration API 完整文档

概述

API navigator.vibrate(pattern)

触发设备振动。pattern 可以是数字（振动毫秒数）或数字数组（振动/暂停交替）。

参数类型

类型 说明 示例
number振动指定毫秒数200
number[]振动/暂停交替模式[200, 100, 200]

返回值

RETURN boolean

返回 true 表示振动成功，false 表示失败或不支持。

停止振动

METHOD navigator.vibrate(0) 或 navigator.vibrate([])

传入 0 或空数组可以停止当前振动。

使用示例

// 检测支持 if ('vibrate' in navigator) { // 简单振动 200ms navigator.vibrate(200); // 振动模式: 振动200ms, 暂停100ms, 振动200ms navigator.vibrate([200, 100, 200]); // 停止振动 navigator.vibrate(0); // 或 navigator.vibrate([]); }

注意事项

  • 需要用户交互后才能触发振动（如点击按钮）
  • 桌面浏览器通常不支持此 API
  • 振动模式数组长度不能超过 99
  • 振动时长会被限制在合理范围内
