播放控制 API
player.play() - 播放视频
player.pause() - 暂停视频
player.paused() - 是否暂停
player.ended() - 是否结束
player.seeking() - 是否跳转中
时间控制 API
player.currentTime() - 获取当前时间
player.currentTime(60) - 设置时间
player.duration() - 获取总时长
player.remainingTime() - 剩余时间
player.buffered() - 缓冲区间
player.bufferedPercent() - 缓冲百分比
player.seekable() - 可跳转区间
音量控制 API
player.volume() - 获取音量
player.volume(0.5) - 设置音量
player.muted() - 是否静音
player.muted(true) - 设置静音
全屏/画中画 API
player.requestFullscreen() - 进入全屏
player.exitFullscreen() - 退出全屏
player.isFullscreen() - 是否全屏
player.requestPictureInPicture() - 画中画
player.exitPictureInPicture() - 退出画中画
player.isInPictureInPicture() - 是否画中画
轨道 API
player.textTracks() - 字幕轨道
player.audioTracks() - 音频轨道
player.videoTracks() - 视频轨道
player.addTextTrack(kind, label, lang)
事件 API
player.on('play', fn) - 绑定事件
player.one('ended', fn) - 绑定一次
player.off('play', fn) - 解绑事件
player.trigger('custom') - 触发事件
